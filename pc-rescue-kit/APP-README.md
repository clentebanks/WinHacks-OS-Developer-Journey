# WinHacks PC Rescue Kit — Primera interfaz funcional

## Abrir localmente

No abras `app/index.html` directamente con doble clic. Los módulos JSON requieren un servidor local.

Desde la raíz del proyecto ejecuta uno de estos comandos:

```powershell
npx serve .
```

Después abre la URL que indique la consola y entra en:

```text
/pc-rescue-kit/app/
```

En Netlify funcionará directamente en:

```text
https://tu-dominio/pc-rescue-kit/app/
```
