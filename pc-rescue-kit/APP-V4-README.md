# PC Rescue Kit App v4

## Nuevo diagnóstico

Esta versión agrega el flujo completo:

```text
Virus o malware
```

Incluye:

- Historial de protección de Defender
- Examen rápido
- Examen completo
- Examen sin conexión
- Microsoft MRT
- Cuatro reglas de decisión
- Cuatro validaciones finales
- Avisos especiales antes de reiniciar o restaurar elementos

## Después de reemplazar la carpeta

```powershell
npm run knowledge:validate
npm run knowledge:index
npx serve .
```

Abre:

```text
http://localhost:3000/pc-rescue-kit/app/
```
