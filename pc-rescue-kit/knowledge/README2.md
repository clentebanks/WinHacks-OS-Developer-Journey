# Flujo inicial: PC lenta

Copia la carpeta `pc-rescue-kit` sobre la existente.

Luego ejecuta:

```powershell
npm run knowledge:validate
npm run knowledge:index
npm run knowledge:test
```

Resultado esperado:

- procedures: 4
- symptoms: 1
- questions: 1
- rules: 1
- validations: 1
- totalItems: 8
