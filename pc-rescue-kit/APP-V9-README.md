# PC Rescue Kit App v9 — Dashboard + respaldo

## Funciones nuevas

- Dashboard de sesiones.
- Total, finalizadas, pendientes y WHS promedio.
- Ranking de problemas diagnosticados.
- Exportar todas las sesiones en JSON.
- Importar y combinar o reemplazar el historial.
- Metadatos SEO, Open Graph y Schema SoftwareApplication.
- Mejoras responsive para dashboard y copias.

## Prueba

```powershell
npm run knowledge:validate
npm run knowledge:index
npx serve .
```

Abre:

```text
http://localhost:3000/pc-rescue-kit/app/
```
