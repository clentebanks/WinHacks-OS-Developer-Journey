# PC Rescue Kit App v2

Esta versión completa las cuatro respuestas del flujo **PC lenta**:

- Disco elevado
- Memoria RAM elevada
- CPU elevada
- No sé qué recurso está elevado

Después de copiar la carpeta sobre la existente, ejecuta:

```powershell
npm run knowledge:validate
npm run knowledge:index
npm run decision:test
```

Para probar la aplicación:

```powershell
npx serve .
```

Abre:

```text
/pc-rescue-kit/app/
```
