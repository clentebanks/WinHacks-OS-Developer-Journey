# 1. Propósito del documento

Este documento define la identidad oficial de WinHacks.

Todo contenido, diseño, artículo, Short, PDF, curso o herramienta desarrollada para el proyecto debe respetar las reglas establecidas aquí.

Si existe un conflicto entre un diseño nuevo y este documento, prevalece este documento hasta que se publique una nueva versión.

---

# 2. ¿Qué es WinHacks?

WinHacks es una plataforma educativa especializada en Windows 10 y Windows 11.

Su objetivo es enseñar funciones ocultas, herramientas integradas, comandos, optimización, seguridad y productividad mediante contenido práctico y fácil de entender.

WinHacks no busca enseñar informática avanzada por sí misma; busca ayudar a que cualquier usuario aproveche mejor Windows utilizando herramientas que ya vienen incluidas en el sistema operativo.

---

# 3. Misión

Ayudar a millones de usuarios a descubrir el verdadero potencial de Windows mediante contenido claro, práctico, seguro y basado en experiencias reales.

---

# 7. Personalidad de la marca

WinHacks debe transmitir:

* Profesional
* Tecnológico
* Moderno
* Cercano
* Curioso
* Práctico
* Confiable

Nunca debe transmitir:

* Arrogancia
* Alarmismo
* Sensacionalismo
* Complejidad innecesaria

---

# 8. Tono de comunicación

Siempre:

* Claro
* Directo
* Respetuoso
* Cercano
* Basado en experiencia

Siempre que sea natural, utilizar experiencias reales.

Ejemplo:

"Siempre veía estas flechas en Windows hasta que descubrí cómo quitarlas."

En lugar de:

"Los expertos no quieren que sepas esto."

---

# 9. Filosofía de contenido

Cada contenido debe cumplir al menos uno de estos objetivos:

* Descubrir una función oculta.
* Resolver un problema real.
* Optimizar Windows.
* Mejorar la seguridad.
* Aumentar la productividad.
* Enseñar una herramienta integrada.

Si no cumple alguno de estos objetivos, probablemente no pertenece al ecosistema WinHacks.

# 14. Estilo fotográfico

Todas las imágenes deben seguir una misma línea.

* Fondo oscuro
* Alto contraste
* Iluminación azul
* Estética tecnológica
* Windows como protagonista

---

# 15. Miniaturas

Objetivo principal:

Detener el desplazamiento del usuario.

Reglas:

* Máximo cuatro palabras.
* Un único punto de atención.
* Mucho contraste.
* Texto muy grande.
* Fondo limpio.
* Evitar saturación de elementos.

---

# 16. Shorts

Todo Short debe:

* Captar la atención en menos de dos segundos.
* Presentar primero el problema o la curiosidad.
* Mostrar una solución práctica.
* Basarse, cuando sea apropiado, en una experiencia personal.
* Invitar a seguir aprendiendo.

---

# 19. WinHacks Academy

La Academy representa el siguiente nivel de aprendizaje.

Nunca sustituye al contenido gratuito.

El contenido gratuito debe aportar valor suficiente por sí mismo.

---

# 20. Calidad mínima

Antes de publicar cualquier contenido verificar:

* Información comprobada.
* Ortografía.
* SEO completo.
* Compatibilidad móvil.
* Enlaces funcionales.
* Formularios funcionando.
* Recursos descargables disponibles.
* Integración con YouTube.

---

# 21. Lo que WinHacks nunca hará

* Publicar contenido falso.
* Recomendar software inseguro.
* Copiar contenido de otros autores.
* Publicar comandos sin explicar qué hacen.
* Omitir advertencias de seguridad.
* Priorizar clics sobre la confianza del usuario.

---

# 22. Principios editoriales

Toda publicación debe responder al menos una pregunta útil del usuario.

Todo tutorial debe explicar:

* Qué hace.
* Para qué sirve.
* Cuándo utilizarlo.
* Cómo revertir el cambio si es necesario.

---

# 23. Identidad de marca

WinHacks no pretende impresionar con tecnicismos.

Pretende que cualquier persona termine un tutorial pensando:

> "No sabía que Windows podía hacer eso."

Ese sentimiento de descubrimiento debe estar presente en la mayor parte del contenido.

---

# 24. Manifiesto WinHacks

Windows es una herramienta mucho más poderosa de lo que la mayoría imagina.

Millones de personas utilizan cada día funciones básicas mientras decenas de herramientas avanzadas permanecen ocultas.

WinHacks existe para descubrir esas funciones, explicarlas con claridad y ayudar a que cualquier usuario aproveche mejor su computadora sin instalar programas innecesarios.

Cada artículo, cada Short, cada guía y cada curso debe reforzar esa misión.

La confianza del usuario es el activo más importante de WinHacks y nunca será sacrificada por conseguir más clics o más visitas.
