# Glossary

## CLI

Command-line interface. A text-based way to run WinHacks OS operations.

## Core

Shared services and contracts that feature modules can use without depending on a user interface.

## Characterization test

A test that records existing behavior before refactoring, including behavior that may not yet be ideal.

## Legacy code

Existing code retained for reference or migration. The term does not mean the code has no value.

## Workspace

An npm repository that manages multiple packages or applications from one root.
