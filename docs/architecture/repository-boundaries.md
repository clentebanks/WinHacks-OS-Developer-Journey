# Repository Boundaries

## Production WinHacks Repository

Contains the public website, articles, images, downloads, Netlify configuration and production deployment history.

## WinHacks OS Developer Journey

Contains learning material, architecture, experiments, reusable packages, applications, tests and a cleaned reference snapshot of the automation engine.

## Boundary Rule

Nothing in this repository may automatically change the production repository until an explicit integration workflow is designed, reviewed, tested and documented.
