El truco secreto para crear códigos QR desde Windows

Windows puede generar códigos QR directamente desde CMD o Terminal usando curl. Es perfecto para compartir links, redes sociales, recursos técnicos, enlaces de WiFi, formularios o tu canal de YouTube sin instalar programas adicionales.

Mira cómo crear un QR desde Windows

En este Short te muestro cómo generar un código QR desde CMD o Terminal de Windows usando un comando rápido con curl.

Qué aprenderás

  • Cómo crear un código QR desde Windows usando la terminal.
  • Cómo usar curl para generar un QR con cualquier link.
  • Cómo guardar el código QR como imagen PNG.
  • Cómo usar este truco para técnicos, estudiantes y creadores de contenido.

¿Qué hace este comando?

Este método usa curl, una herramienta de línea de comandos incluida en versiones modernas de Windows, para solicitar un código QR a un servicio web y guardarlo como imagen.

El resultado es un archivo PNG que puedes escanear con tu teléfono, insertar en una presentación, enviar a un cliente o usar en tus publicaciones.

💡 Consejo WinHacks

Crea una carpeta llamada QR dentro de Documentos o Escritorio. Así podrás guardar todos tus códigos QR organizados por proyecto, cliente o red social.

🚨

Antes de compartir el QR

Escanea siempre el código antes de publicarlo. Si el enlace tiene un error, el QR enviará a una página equivocada y tendrás que volver a generarlo.

Paso a paso

1

Abre CMD o Terminal

Presiona Win, busca CMD o Terminal y abre la consola de Windows.

2

Prepara el enlace

Copia el enlace que quieres convertir en código QR. Puede ser una página web, tu canal de YouTube, una red social, un formulario o cualquier URL pública.

3

Ejecuta el comando curl

Usa este comando como base. Cambia el enlace por la URL que quieres convertir:

curl "https://quickchart.io/qr?text=https://winhacks.dev" --output qr-winhacks.png
4

Revisa el archivo generado

Cuando termine el comando, Windows creará el archivo qr-winhacks.png en la carpeta actual de la terminal.

5

Abre el código QR

Haz doble clic sobre el archivo PNG para abrirlo. También puedes moverlo a otra carpeta, insertarlo en un PDF, usarlo en una presentación o compartirlo con otra persona.

6

Escanea el QR antes de compartirlo

Usa tu teléfono para escanear el código QR y confirma que abre el enlace correcto. Este paso evita errores antes de publicarlo o enviarlo.

Ejemplo para un canal de YouTube

Si quieres crear un QR para tu canal, cambia el valor después de text= por el enlace real de YouTube.

curl "https://quickchart.io/qr?text=https://youtube.com/@WinHacksOficial" --output qr-youtube.png

Ejemplo para compartir WiFi

También puedes crear un QR para una red WiFi usando el formato estándar:

curl "https://quickchart.io/qr?text=WIFI:T:WPA;S:NombreWiFi;P:TuClaveWiFi;;" --output qr-wifi.png
🚨

Cuidado con QR de WiFi

No publiques un QR de WiFi en redes sociales ni en capturas públicas. Ese código puede contener el nombre y la contraseña de tu red.

Creadores de contenido

Puedes generar QR para tu canal, newsletter, recursos gratuitos, página web, formulario de contacto o redes sociales.

Técnicos de PC

Puedes crear QR para manuales, páginas de soporte, drivers, formularios, instrucciones rápidas o redes WiFi autorizadas.

Puedes usar QR en presentaciones, tareas, documentos compartidos, materiales de clase o enlaces de proyectos.

✅ Resultado esperado

Al ejecutar el comando, Windows descargará una imagen PNG con tu código QR. Ese archivo quedará listo para escanear, compartir o insertar en documentos.

Errores comunes

El comando dice que curl no existe

Las versiones modernas de Windows 10 y Windows 11 incluyen curl de forma predeterminada. Si recibes un error, actualiza Windows o verifica escribiendo:

curl --version

No encuentro el archivo PNG

El código QR se guarda en la carpeta donde ejecutaste el comando. Si no sabes cuál es, escribe:

cd

También puedes abrir el Explorador de archivos desde esa ubicación usando:

explorer .

El QR abre una página incorrecta

Revisa cuidadosamente el enlace que escribiste después de text=. Un solo carácter incorrecto hará que el código QR apunte a otra dirección.

No tengo conexión a Internet

Este método utiliza un servicio web para generar el código QR, por lo que es necesario tener acceso a Internet mientras ejecutas el comando.

Preguntas frecuentes

¿Necesito instalar algún programa?

No. Windows 10 y Windows 11 ya incluyen curl en la mayoría de versiones modernas.

¿Funciona en Windows 11?

Sí. Puedes ejecutarlo desde CMD, PowerShell o Windows Terminal.

¿Puedo crear QR para cualquier enlace?

Sí. Puedes generar códigos QR para páginas web, formularios, perfiles sociales, documentos públicos, canales de YouTube o cualquier URL válida.

¿Puedo cambiar el nombre del archivo?

Sí. Solo reemplaza qr-winhacks.png por el nombre que desees al final del comando.
