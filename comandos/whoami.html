¿Qué hace whoami?

whoami muestra el contexto de usuario actual. Normalmente devuelve el nombre del equipo o dominio seguido del usuario activo.

Es útil cuando necesitas confirmar si estás usando una cuenta local, una cuenta de dominio, una sesión administrativa o un usuario específico dentro de Windows.

💡 Consejo WinHacks

Combina whoami con hostname para saber rápidamente en qué equipo estás trabajando y con qué usuario.

⚠️

Importante

Este comando solo muestra información. No cambia permisos, no modifica usuarios y no requiere privilegios de administrador.

Paso a paso

1

Abre CMD

Presiona Win, escribe CMD y abre el Símbolo del sistema.

2

Ejecuta el comando

Escribe este comando y presiona Enter.

whoami
3

Interpreta el resultado

Verás un formato parecido a equipo\usuario o dominio\usuario.

4

Usa whoami en PowerShell

El mismo comando también funciona desde PowerShell y Terminal de Windows.

whoami
5

Combínalo con hostname

Para identificar equipo y usuario al mismo tiempo, usa:

hostname && whoami
6

Consulta información completa

Para obtener más contexto del equipo, puedes combinarlo con:

systeminfo

¿Cuándo utilizar whoami?

  • Confirmar qué usuario está activo.
  • Verificar sesiones en soporte remoto.
  • Identificar cuentas locales o de dominio.
  • Usarlo en scripts de CMD o PowerShell.
  • Diagnosticar permisos de usuario.
  • Comprobar contexto antes de ejecutar comandos técnicos.

✅ Resultado esperado

Windows mostrará el usuario actual y su contexto, normalmente con el formato equipo\usuario o dominio\usuario.

Errores comunes

El comando no se reconoce

Verifica que escribiste correctamente whoami. Está disponible por defecto en Windows.

El usuario no es el esperado

Puede que estés usando otra cuenta, sesión remota o usuario administrador diferente. Revisa el perfil activo en Windows.

No muestra el nombre completo

El comando muestra el identificador de usuario, no necesariamente el nombre visible completo de la cuenta.

No sé si es cuenta local o dominio

Si aparece el nombre del equipo antes de la barra invertida, suele ser cuenta local. Si aparece un dominio, estás en un entorno administrado.

Preguntas frecuentes

¿whoami cambia algo en Windows?

No. Solo muestra información del usuario actual.

¿Funciona en Windows 11?

Sí. También funciona en Windows 10 y Windows Server.

¿Necesito permisos de administrador?

No. Cualquier usuario puede ejecutarlo.

¿Cuál es la diferencia entre whoami y hostname?

whoami muestra el usuario actual. hostname muestra el nombre del equipo.
