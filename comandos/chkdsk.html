¿Cómo revisar errores en un SSD o disco duro en Windows?

Para revisar errores en tu SSD o disco duro, abre CMD como administrador y ejecuta chkdsk C: /scan. Este comando analiza la unidad C en línea y detecta problemas del sistema de archivos. Si Windows encuentra errores, puedes programar una reparación con chkdsk C: /f o una revisión profunda con chkdsk C: /f /r, pero antes conviene hacer copia de seguridad.

Importante antes de reparar

Si sospechas que tu SSD o disco duro está fallando físicamente, primero haz copia de seguridad. Las opciones de reparación como /f y especialmente /r pueden tardar mucho tiempo y forzar lectura intensiva del disco. No las uses a ciegas en un disco con síntomas graves.

Qué es CHKDSK

CHKDSK significa Check Disk. Es una herramienta integrada de Windows que revisa la integridad del sistema de archivos de una unidad. Puede detectar errores lógicos, problemas de índices, entradas dañadas y sectores defectuosos que impiden leer datos correctamente.

💡 Corrección importante

CHKDSK no es lo mismo que una prueba completa de salud del SSD. Para revisar desgaste, vida útil o datos S.M.A.R.T., también debes usar herramientas como el software del fabricante del SSD o comandos de PowerShell.

Comandos principales

Empieza siempre por el comando menos invasivo:

chkdsk C: /scan

Si Windows informa errores y necesitas corregirlos, usa:

chkdsk C: /f

Para una revisión profunda que intenta localizar sectores dañados y recuperar información legible:

chkdsk C: /f /r
⚠️

No empieces con /r sin necesidad

/r puede tardar horas en discos grandes. En SSD, normalmente es mejor empezar con /scan y revisar también la salud S.M.A.R.T. antes de lanzar una revisión profunda.

Paso a paso

1

Haz copia de seguridad si sospechas fallo real

Si notas archivos corruptos, pantallazos, congelamientos, sonidos extraños en un disco duro mecánico o errores repetidos, copia primero tus archivos importantes a otra unidad o a la nube.

2

Abre CMD como administrador

Busca CMD en Inicio, haz clic derecho y selecciona Ejecutar como administrador. También puedes usar Terminal de Windows como administrador.

3

Ejecuta una revisión segura

Para revisar la unidad C sin iniciar una reparación completa, ejecuta:

chkdsk C: /scan

Windows analizará la unidad y te dirá si encontró errores o si no se requieren acciones adicionales.

4

Interpreta el resultado

Si aparece un mensaje indicando que no se encontraron problemas, la estructura del sistema de archivos está bien. Si aparecen errores, Windows puede sugerir una reparación.

✅ Resultado esperado

Un resultado limpio debe indicar que Windows examinó el sistema de archivos y no encontró problemas.

5

Repara errores si Windows los detecta

Si CHKDSK detecta errores del sistema de archivos, puedes ejecutar:

chkdsk C: /f

En la unidad del sistema, Windows puede pedir programar la reparación para el próximo reinicio. Acepta solo si ya guardaste tu trabajo.

6

Usa revisión profunda solo cuando sea necesario

Si hay sospecha de sectores dañados o errores graves, puedes usar:

chkdsk C: /f /r

Este proceso puede tardar bastante. No apagues el equipo mientras se ejecuta.

Qué significa cada parámetro

/scan

Analiza la unidad en línea. Es el mejor primer paso porque detecta errores sin iniciar una reparación agresiva.

/f

Corrige errores del sistema de archivos. Puede requerir reiniciar si la unidad está en uso.

/r

Localiza sectores dañados e intenta recuperar información legible. Incluye la función de /f y puede tardar mucho más.

Cuándo usar este comando

  • Windows muestra errores de disco.
  • La PC se apagó mal o hubo corte de energía.
  • Archivos aparecen corruptos o no se pueden abrir.
  • El sistema se congela al acceder a una carpeta o unidad.
  • El Explorador tarda demasiado en abrir una unidad.

Cuándo NO usarlo como primera opción

  • Si el disco hace ruidos físicos extraños. Primero respalda.
  • Si el SSD desaparece del BIOS o de Windows. Revisa conexión y salud del dispositivo.
  • Si tienes datos críticos sin copia. Primero clona o respalda.
  • Si solo quieres medir vida útil del SSD. Usa S.M.A.R.T. o herramienta del fabricante.

Errores comunes

Ejecutar CHKDSK sin administrador

Puedes ver información básica, pero para reparar correctamente necesitas permisos de administrador.

Usar /r en un SSD sin entenderlo

No es que esté prohibido, pero puede ser innecesario y lento. Primero usa /scan y revisa la salud del SSD.

No hacer respaldo antes de reparar

Si el disco está fallando físicamente, cualquier operación intensiva puede empeorar la situación. Protege tus datos primero.

Confundir errores del sistema de archivos con vida útil del SSD

Un SSD puede tener sistema de archivos correcto y aun así estar desgastado. CHKDSK y S.M.A.R.T. se complementan.

Preguntas frecuentes

¿CHKDSK funciona en Windows 11?

Sí. CHKDSK está disponible en Windows 11 y puede ejecutarse desde CMD, PowerShell o Terminal.

¿CHKDSK funciona en Windows 10?

Sí. También funciona en Windows 10 y usa parámetros como /scan, /f y /r.

¿Qué comando es más seguro para empezar?

Empieza con chkdsk C: /scan. Es una revisión en línea y no inicia una reparación profunda.

¿Puedo revisar otra unidad que no sea C?

Sí. Cambia la letra. Por ejemplo: chkdsk D: /scan o chkdsk E: /scan.

¿Por qué CHKDSK se ejecuta al reiniciar?

Porque Windows no puede reparar completamente la unidad del sistema mientras está en uso.

¿CHKDSK mejora el rendimiento?

No es una herramienta de optimización. Puede ayudar si el bajo rendimiento se debe a errores del sistema de archivos.

¿CHKDSK reemplaza CrystalDiskInfo?

No. CHKDSK revisa el sistema de archivos. CrystalDiskInfo u otras herramientas S.M.A.R.T. revisan datos de salud del disco.

¿Puedo cancelar CHKDSK?

No es recomendable interrumpirlo durante una reparación. Si solo está escaneando, espera a que termine para evitar inconsistencias.

¿Qué hago si CHKDSK encuentra muchos errores?

Haz copia de seguridad inmediatamente, revisa la salud S.M.A.R.T. del disco y considera reemplazarlo si los errores se repiten.
