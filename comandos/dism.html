DISM: el comando técnico para reparar la imagen de Windows

DISM es una herramienta avanzada de Windows que permite comprobar y reparar la imagen del sistema. Es especialmente útil cuando sfc /scannow no puede reparar archivos dañados o cuando Windows presenta errores después de actualizaciones, apagados forzados o corrupción interna.

¿Qué es DISM?

DISM significa Deployment Image Servicing and Management. En Windows 10 y Windows 11 se usa para revisar y reparar la imagen del sistema, especialmente el almacén de componentes que Windows utiliza para recuperar archivos dañados.

Cuando sfc /scannow no puede corregir algunos errores, DISM puede reparar la fuente interna que SFC utiliza para restaurar archivos protegidos del sistema.

💡 Consejo WinHacks

La secuencia técnica recomendada cuando Windows está dañado es: DISM /Online /Cleanup-Image /RestoreHealth y después sfc /scannow.

⚠️

Importante

Ejecuta DISM desde CMD o PowerShell como administrador. No cierres la ventana aunque parezca detenido en un porcentaje durante varios minutos.

Paso a paso

1

Abre CMD como administrador

Presiona Win, escribe CMD, haz clic derecho y selecciona Ejecutar como administrador.

2

Comprueba la imagen de Windows

Primero puedes comprobar si la imagen tiene corrupción detectable.

DISM /Online /Cleanup-Image /CheckHealth
3

Analiza con más detalle

Si quieres una comprobación más profunda, ejecuta:

DISM /Online /Cleanup-Image /ScanHealth
4

Repara la imagen del sistema

Para reparar componentes dañados, ejecuta el comando principal:

DISM /Online /Cleanup-Image /RestoreHealth
5

Espera hasta el 100%

El proceso puede quedarse varios minutos en 20%, 40% o 62.3%. No cierres CMD. Espera a que termine completamente.

6

Ejecuta SFC después de DISM

Cuando DISM termine, ejecuta SFC para reparar archivos protegidos usando la imagen corregida.

sfc /scannow

Comandos útiles de DISM

Comprobar corrupción rápida

DISM /Online /Cleanup-Image /CheckHealth

Analizar imagen del sistema

DISM /Online /Cleanup-Image /ScanHealth

Reparar imagen de Windows

DISM /Online /Cleanup-Image /RestoreHealth

Reparar archivos protegidos después

sfc /scannow

¿Cuándo utilizar DISM?

  • Cuando sfc /scannow no puede reparar archivos dañados.
  • Después de actualizaciones fallidas.
  • Cuando Windows presenta errores extraños.
  • Cuando funciones del sistema no abren.
  • Después de apagados forzados o corrupción del sistema.
  • Antes de pensar en reinstalar Windows.

✅ Resultado esperado

DISM comprobará y reparará la imagen de Windows. Después, SFC tendrá una fuente más confiable para reparar archivos protegidos del sistema.

Errores comunes

DISM se queda en un porcentaje

Es normal que parezca detenido durante varios minutos. No cierres la ventana; espera a que el proceso termine.

Error 0x800f081f

Windows no pudo encontrar los archivos fuente necesarios. En casos avanzados puede requerirse una imagen ISO de Windows como origen de reparación.

No tengo conexión a Internet

RestoreHealth puede necesitar Windows Update para descargar componentes. Conecta Internet y vuelve a intentar.

Acceso denegado

Ejecuta CMD o PowerShell como administrador.

Preguntas frecuentes

¿DISM borra mis archivos?

No. DISM repara componentes del sistema. No elimina documentos, fotos ni archivos personales.

¿Debo ejecutar DISM antes o después de SFC?

Si SFC falla, ejecuta DISM primero y luego repite sfc /scannow.

¿Funciona en Windows 11?

Sí. También funciona en Windows 10.

¿Necesito administrador?

Sí. Debes abrir CMD o PowerShell como administrador.
