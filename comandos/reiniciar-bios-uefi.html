En este Short te muestro cómo crear un acceso directo para reiniciar tu PC y entrar directamente al BIOS o UEFI.

Qué aprenderás

  • Cómo entrar al BIOS o UEFI desde Windows.
  • Qué hace el comando shutdown /r /fw /t 0.
  • Cómo crear un acceso directo en el escritorio.
  • Cómo activar “Ejecutar como administrador”.
  • Cuándo usar este método de forma segura.

Qué hace este comando

El comando shutdown /r /fw /t 0 reinicia el equipo y solicita que el siguiente arranque entre directamente a la interfaz de firmware UEFI. Esto evita depender de teclas como F2, DEL, ESC o F10, que cambian según la marca del equipo.

Es especialmente útil cuando necesitas cambiar el orden de arranque, activar TPM, revisar Secure Boot, habilitar virtualización o entrar a opciones avanzadas del firmware.

Consejo WinHacks

Si vas a usar este acceso directo con frecuencia, cambia su icono por uno de chip, placa base o herramienta técnica. Así lo identificas rápido en el escritorio.

Advertencia de seguridad

Este comando reinicia el equipo inmediatamente. Guarda tus archivos abiertos antes de usarlo. El comando no modifica el BIOS por sí solo; cualquier cambio dentro del firmware depende de lo que tú selecciones después del reinicio.

Qué significa cada parámetro

  • shutdown: ejecuta la herramienta de apagado o reinicio de Windows.
  • /r: indica que el equipo debe reiniciarse.
  • /fw: solicita que el próximo arranque entre al firmware UEFI.
  • /t 0: reinicia inmediatamente, sin esperar segundos.

Paso a paso para crear el acceso directo

1

Haz clic derecho en el escritorio

En una zona vacía del escritorio, haz clic derecho y selecciona Nuevo > Acceso directo.

2

Escribe el comando

En el campo de ubicación del elemento, pega este comando:

shutdown /r /fw /t 0
3

Asigna un nombre

Puedes nombrarlo como Reiniciar al BIOS, Abrir UEFI o Entrar al firmware.

4

Activa ejecutar como administrador

Haz clic derecho sobre el acceso directo, entra en Propiedades, luego Avanzado y marca Ejecutar como administrador.

5

Cambia el icono

En las propiedades del acceso directo, usa la opción Cambiar icono. Esto no es obligatorio, pero hace que el acceso directo se vea más profesional.

6

Ejecuta el acceso directo

Al abrirlo como administrador, Windows reiniciará el equipo y debería enviarte directamente al BIOS o a la pantalla de configuración UEFI.

Cuándo conviene usar este método

Este truco es útil cuando necesitas entrar al firmware para cambiar opciones que no están dentro de Windows.

  • Cambiar el orden de arranque.
  • Activar TPM.
  • Activar o revisar Secure Boot.
  • Habilitar virtualización Intel VT-x o AMD-V.
  • Preparar una instalación de Windows desde USB.
  • Revisar configuración avanzada del equipo.

No funciona en todos los equipos

Este comando está pensado para equipos con firmware UEFI. Si tu PC usa BIOS Legacy, si el fabricante bloquea el acceso o si Windows no tiene permisos suficientes, el reinicio puede ocurrir sin entrar al firmware.

Resultado esperado

Después de ejecutar el acceso directo como administrador, tu PC debe reiniciarse y abrir directamente la configuración del BIOS o UEFI, sin presionar teclas durante el arranque.

Errores comunes

No se abre el BIOS después de reiniciar

Puede ocurrir si tu equipo no usa UEFI, si el comando no se ejecutó como administrador o si el firmware del fabricante no acepta esta instrucción desde Windows.

El acceso directo pide permisos

Es normal. Este comando necesita permisos elevados porque reinicia el sistema y solicita entrada al firmware. Activa Ejecutar como administrador en las propiedades del acceso directo.

Se reinició demasiado rápido

Eso pasa porque el parámetro /t 0 indica reinicio inmediato. Si quieres unos segundos para prepararte, puedes usar /t 10 en lugar de /t 0.

Preguntas frecuentes

¿Este comando daña el BIOS?

No. El comando solo solicita reiniciar hacia el firmware UEFI. No cambia configuraciones por sí solo.

¿Funciona en Windows 10 y Windows 11?

Sí, puede funcionar en Windows 10 y Windows 11 cuando el equipo usa UEFI y el comando se ejecuta con permisos adecuados.

¿Necesito presionar F2, DEL o ESC?

No debería ser necesario. La ventaja de este método es reiniciar directamente al firmware sin depender de la tecla específica del fabricante.

¿Qué hago si mi PC no entra al BIOS?

Prueba ejecutarlo como administrador. Si sigue sin funcionar, revisa si tu equipo usa UEFI o entra desde Configuración > Sistema > Recuperación > Inicio avanzado.
