¿Qué hace driverquery?

driverquery es un comando integrado de Windows que muestra información sobre drivers instalados o cargados en el sistema. Es útil para técnicos, diagnóstico, revisión de módulos y detección básica de conflictos.

Puede mostrar el nombre del módulo, tipo de controlador, estado, fecha de vínculo, ruta y otros detalles según el formato que uses.

💡 Consejo WinHacks

Usa driverquery /v si quieres más detalles. Para revisar con calma, exporta el resultado a un archivo TXT o CSV.

🚨

No elimines drivers al azar

Este comando es para revisar información. Si ves un driver extraño, investígalo primero. Eliminar o desactivar drivers sin saber puede causar errores, pérdida de red, audio, pantalla o fallos del sistema.

Paso a paso

1

Abre CMD

Presiona Win, escribe CMD y abre el Símbolo del sistema. Para listar drivers básicos, normalmente no necesitas permisos de administrador.

2

Ejecuta driverquery

Escribe este comando y presiona Enter:

driverquery
3

Revisa la lista de drivers

CMD mostrará columnas como Module Name, Display Name, Driver Type y Link Date.

4

Usa el modo avanzado

Para ver más información técnica, ejecuta:

driverquery /v
5

Exporta el resultado

Si la lista es muy larga, guárdala en un archivo para revisarla mejor:

driverquery /v > drivers.txt
6

Exporta como CSV

Para abrir la lista en Excel o una hoja de cálculo, usa formato CSV:

driverquery /fo csv /v > drivers.csv

Comandos útiles con driverquery

Lista básica

driverquery

Modo avanzado

driverquery /v

Formato tabla

driverquery /fo table

Formato CSV

driverquery /fo csv /v > drivers.csv

✅ Resultado esperado

CMD mostrará una lista de drivers del sistema. Con /v verás información más avanzada, y con /fo csv podrás exportarla para análisis técnico.

Errores comunes

La lista es demasiado larga

Exporta el resultado a TXT o CSV para revisarlo mejor:

driverquery /v > drivers.txt

No entiendo los nombres

Muchos drivers usan nombres técnicos. No asumas que algo es peligroso solo porque no reconoces el nombre. Investiga antes de tomar acción.

Quiero borrar un driver sospechoso

No uses esta lista para eliminar cosas a ciegas. Primero identifica el fabricante, el dispositivo asociado y si pertenece a Windows, GPU, red, audio o seguridad.

Preguntas frecuentes

¿driverquery modifica Windows?

No. Solo muestra información. Es un comando de consulta, no de reparación ni eliminación.

¿Funciona en Windows 11?

Sí. También funciona en Windows 10.

¿Necesito administrador?

Para la lista básica normalmente no. Para revisiones más completas, conviene abrir CMD como administrador.

¿Esto detecta drivers malos automáticamente?

No. Muestra información. La detección de conflictos requiere interpretar fechas, fabricantes, eventos del sistema y síntomas del equipo.
