# Developer Roadmap

The roadmap introduces each technology immediately before it is required by WinHacks OS.

## Level 0 — Development environment

**Outcome:** A safe, repeatable local environment.

Learn:

- Files and folders
- Terminal basics
- VS Code
- Node.js and npm
- Environment variables
- Repository validation

Deliverable: complete `labs/00-foundation`.

## Level 1 — Git and GitHub

**Outcome:** Work without losing changes and publish a professional history.

Learn:

- Repository, commit, branch, merge
- `.gitignore`
- Conventional Commits
- Issues and pull requests
- Basic conflict resolution

Deliverable: first feature branch and reviewed pull request.

## Level 2 — JavaScript

**Outcome:** Read and modify the existing engine confidently.

Learn:

- Values, variables, conditions, loops
- Functions
- Arrays and objects
- Errors
- Modules
- Asynchronous programming
- Filesystem basics

Deliverable: repository-status script with tests.

## Level 3 — Node.js

**Outcome:** Build useful command-line tools.

Learn:

- Runtime and process
- Filesystem and paths
- Command-line arguments
- npm scripts
- package boundaries
- errors and exit codes

Deliverable: `winhacks doctor`.

## Level 4 — TypeScript

**Outcome:** Define stable contracts between modules.

Learn:

- Primitive and structural types
- interfaces and type aliases
- unions and narrowing
- generics
- compiler configuration
- migration from JavaScript

Deliverable: typed CLI and core package.

## Level 5 — Testing

**Outcome:** Refactor existing code without changing behavior accidentally.

Learn:

- unit, integration, and characterization tests
- fixtures
- assertions
- test doubles
- coverage as a diagnostic

Deliverable: tests around the first legacy module.

## Level 6 — React

**Outcome:** Build reusable user-interface components.

Deliverable: local report viewer.

## Level 7 — Next.js

**Outcome:** Connect UI, server logic, and internal APIs.

Deliverable: dashboard application shell.

## Level 8 — Tailwind CSS

**Outcome:** Implement a consistent, responsive UI system.

Deliverable: WinHacks dashboard design foundation.

## Level 9 — SQLite

**Outcome:** Store configuration, job history, content metadata, and reports.

Deliverable: initial schema and repository layer.

## Level 10 — REST APIs

**Outcome:** Expose stable operations to the dashboard and future integrations.

Deliverable: internal API for jobs and reports.

## Level 11 — Architecture and release

**Outcome:** Produce a documented, tested version 1.0.

Deliverable: stable WinHacks OS release.
