Qué incluye este checklist

  • Revisión de Windows Defender.

  • Verificación de Windows Update.
  • Chequeo rápido de aplicaciones desconocidas.

La mayoría nunca revisa esto

Muchos usuarios creen que su computadora está protegida, pero nunca verifican configuraciones básicas que pueden marcar la diferencia frente a malware, accesos no autorizados o software sospechoso.

Este checklist está diseñado para revisar lo esencial sin instalar programas externos ni tocar configuraciones peligrosas.

💡 Consejo WinHacks

Haz esta revisión después de instalar programas nuevos, conectar dispositivos desconocidos o notar comportamientos raros en Windows.

⚠️

Seguridad sin paranoia

No se trata de activar todo sin entender. La idea es comprobar lo básico: protección activa, actualizaciones, firewall y herramientas integradas.

Elementos incluidos

Windows Defender

Verifica que la protección en tiempo real esté activada y que no haya alertas pendientes.

Firewall

Confirma que el firewall esté habilitado para redes privadas y públicas.

Historial de protección

Revisa amenazas detectadas recientemente, elementos bloqueados o acciones pendientes.

MRT

Ejecuta la herramienta oficial de eliminación de software malintencionado incluida en Windows.

Control de cuentas

Verifica que el Control de cuentas de usuario no esté desactivado.

Windows Update

Comprueba que el sistema tenga actualizaciones importantes instaladas.

Preguntas frecuentes

¿Este checklist es gratis?

Sí. Forma parte de la biblioteca gratuita de WinHacks.

¿Funciona en Windows 11?

Sí. Está pensado para Windows 11 y también para Windows 10.

¿Necesito instalar antivirus externo?

No necesariamente. Este checklist se enfoca en revisar las herramientas integradas de Windows antes de instalar soluciones adicionales.

¿Cada cuánto debo hacer esta revisión?

Una vez al mes es suficiente para muchos usuarios. También conviene hacerlo después de instalar software nuevo.

¿Esto elimina virus automáticamente?

No. Es una guía de revisión. Para analizar amenazas puedes usar Windows Defender y la herramienta MRT.
