¿Qué hace este comando?

arp -a muestra la tabla ARP almacenada por Windows. Esta tabla relaciona direcciones IP locales con direcciones físicas o MAC detectadas recientemente por tu equipo.

No es un escáner completo de red, pero puede ayudarte a ver dispositivos con los que tu PC ya se comunicó o que Windows tiene registrados temporalmente.

Ejecuta primero ping al router o a otro equipo de la red. Luego usa arp -a para ver si Windows registró su dirección MAC.

⚠️

Antes de continuar

arp -a no muestra necesariamente todos los dispositivos conectados al Wi-Fi. Solo muestra entradas que Windows conoce o ha detectado recientemente.

Paso a paso

1

Abre Windows Terminal o CMD

Haz clic derecho en Inicio y abre Terminal, PowerShell o Símbolo del sistema.

Captura pendiente: Windows Terminal abierto.
2

Ejecuta arp -a

Escribe este comando y presiona Enter:

arp -a
Captura pendiente: resultado del comando arp -a.
3

Identifica las direcciones IP

En la columna de Dirección de Internet verás IPs locales detectadas por Windows, normalmente similares a 192.168.1.x.

4

Revisa las direcciones MAC

En la columna de Dirección física verás direcciones MAC. Estas identifican interfaces de red, como tarjetas Wi-Fi, Ethernet o dispositivos del router.

Captura pendiente: dirección física resaltada.
5

Comprueba el tipo de entrada

La columna Tipo puede mostrar entradas dinámicas o estáticas. Las dinámicas son aprendidas automáticamente por Windows.

Comandos útiles relacionados con arp

Ver la tabla ARP

Usa arp -a para mostrar las IPs y MAC conocidas por Windows.

arp -a

Ver entradas de una IP específica

Puedes consultar una IP concreta usando el comando con la dirección local.

arp -a 192.168.1.1

Eliminar una entrada ARP

Usa este comando solo si sabes lo que estás haciendo. Puede servir para limpiar una entrada antigua o incorrecta.

arp -d 192.168.1.1

Qué debes revisar primero

Revisa si aparece la IP de tu router, normalmente parecida a 192.168.1.1. Si aparece junto a una dirección física, tu equipo logró comunicarse con ese dispositivo.

Si no aparece ninguna entrada útil, puedes hacer primero un ping a tu router y luego ejecutar nuevamente arp -a.

✅ Resultado esperado

Al usar arp -a correctamente podrás ver IPs y direcciones MAC detectadas por Windows, lo que ayuda a diagnosticar comunicación dentro de la red local.

Cuándo usar arp -a

  • Cuando quieres ver dispositivos detectados por tu PC.
  • Cuando necesitas revisar la IP y MAC del router.
  • Cuando estás diagnosticando problemas de red local.
  • Cuando quieres confirmar si Windows reconoce otro equipo.
  • Cuando necesitas comparar una IP con una dirección física.

Errores comunes

Pensar que muestra todos los dispositivos

La tabla ARP no es una lista completa de todo lo conectado al Wi-Fi. Para eso necesitas revisar el panel del router o usar herramientas de escaneo específicas.

Confundir IP con MAC

La dirección IP puede cambiar. La dirección MAC pertenece a la interfaz de red, aunque algunos dispositivos pueden usar MAC aleatoria por privacidad.

Revisar el adaptador equivocado

Si tienes VPN, Ethernet y Wi-Fi activos, revisa cuidadosamente qué interfaz está mostrando cada grupo de resultados.

Preguntas frecuentes

¿Qué significa arp?

ARP significa Address Resolution Protocol. Sirve para relacionar direcciones IP con direcciones físicas dentro de una red local.

¿arp -a muestra quién está conectado a mi Wi-Fi?

No de forma completa. Solo muestra entradas conocidas por Windows. Para ver todos los dispositivos conectados, revisa el panel de administración del router.

¿Necesito permisos de administrador?

Para ejecutar arp -a normalmente no. Para eliminar entradas, puede ser necesario abrir la terminal como administrador.

¿Qué es una dirección MAC?

Es un identificador físico de una interfaz de red, como Wi-Fi o Ethernet. Se muestra con pares de caracteres separados por guiones o dos puntos.

¿Por qué veo muchas direcciones?

Puede ser normal si tu equipo se ha comunicado con el router, otros equipos, dispositivos inteligentes, impresoras o servicios locales de red.
