Mira cómo ajustar tu WiFi en Windows

En este Short te muestro cómo entrar a las opciones avanzadas del adaptador WiFi y revisar Transmit Power y Preferred Band.

  • Cómo reducir microcortes sin instalar programas externos.

¿Qué hacen estos ajustes WiFi?

Algunos adaptadores WiFi permiten modificar parámetros avanzados desde Windows. Dos de los más útiles son Transmit Power y Preferred Band.

Transmit Power controla la potencia de transmisión del adaptador. Preferred Band indica si el adaptador debe preferir redes de 2.4 GHz o 5 GHz cuando ambas están disponibles.

💡 Consejo WinHacks

Para gaming o videollamadas, normalmente conviene usar 5 GHz si estás cerca del router. Si estás lejos o hay muchas paredes, 2.4 GHz puede ser más estable.

🚨

Antes de cambiar la configuración

No todos los adaptadores muestran las mismas opciones. Si no ves Transmit Power o Preferred Band, depende del driver, marca del adaptador o modelo de tu tarjeta WiFi.

Paso a paso

1

Abre Ejecutar

Presiona Win + R para abrir la ventana Ejecutar de Windows.

2

Abre las conexiones de red

Escribe el comando correcto y presiona Enter:

ncpa.cpl
3

Entra en las propiedades del WiFi

Haz clic derecho sobre tu adaptador WiFi y selecciona Propiedades.

4

Abre la configuración del adaptador

Dentro de Propiedades, presiona el botón Configurar. Luego entra en la pestaña Opciones avanzadas.

5

Ajusta Transmit Power

Busca Transmit Power, Potencia de transmisión o una opción similar. Si aparece, colócala en el valor más alto disponible.

6

Ajusta Preferred Band

Busca Preferred Band, Banda preferida o una opción similar. Si estás cerca del router, selecciona Prefer 5GHz.

Configuración recomendada

Si estás cerca del router

Usa Transmit Power: Highest y Preferred Band: Prefer 5GHz. Esta combinación puede mejorar latencia y estabilidad para gaming, streaming o videollamadas.

Si estás lejos del router

Puedes probar 2.4 GHz, porque suele tener más alcance. No siempre tendrá la menor latencia, pero puede ser más estable con paredes o distancia.

Si no ves estas opciones

Actualiza el driver WiFi desde Windows Update, la web del fabricante o el software oficial del equipo. Algunos drivers básicos ocultan opciones avanzadas.

✅ Resultado esperado

Después de aplicar estos cambios, tu conexión WiFi puede sentirse más estable, con menos microcortes y una latencia más consistente en juegos, streaming o uso diario.

Errores comunes

Escribí ncpl.cpl y no funciona

El comando correcto es ncpa.cpl. La letra a es importante porque abre las conexiones de red de Windows.

No aparece Transmit Power

Tu adaptador o driver puede no incluir esa opción. Actualiza el controlador WiFi o revisa si aparece con otro nombre como Potencia de transmisión.

No aparece Preferred Band

Algunos adaptadores solo muestran esta opción si son compatibles con doble banda. Si tu tarjeta WiFi solo trabaja en 2.4 GHz, no podrás seleccionar 5 GHz.

Preguntas frecuentes

¿Esto aumenta los megas de mi Internet?

No. Este ajuste no aumenta la velocidad contratada. Puede mejorar estabilidad, latencia y consistencia si el adaptador estaba usando una configuración poco óptima.

¿Sirve para gaming?

Sí puede ayudar, especialmente si el problema era inestabilidad WiFi o cambios de banda. Para gaming competitivo, una conexión Ethernet sigue siendo la opción más estable.

¿Es seguro poner Transmit Power al máximo?

Sí. Es una opción del propio driver. El consumo puede subir ligeramente, pero en laptops o PCs conectadas a corriente suele ser una configuración razonable para estabilidad.

¿Funciona en Windows 11?

Sí. También funciona en Windows 10, siempre que el adaptador WiFi y su driver muestren estas opciones avanzadas.
