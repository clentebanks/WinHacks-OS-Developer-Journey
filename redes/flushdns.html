flushdns: limpia la caché DNS de Windows en segundos

Windows guarda respuestas DNS en caché para abrir sitios web más rápido. Pero cuando esa caché queda desactualizada o corrupta, algunas páginas pueden fallar, cargar mal o apuntar a una dirección incorrecta. Con ipconfig /flushdns puedes limpiar esa caché desde CMD.

¿Qué hace ipconfig /flushdns?

ipconfig /flushdns borra la caché local de resolución DNS de Windows. Esa caché guarda respuestas recientes de dominios para no tener que consultar el servidor DNS cada vez que visitas una página.

Si un sitio cambió de servidor, si tu DNS respondió mal o si Windows guardó una respuesta antigua, limpiar la caché puede forzar una nueva consulta DNS y resolver errores de navegación.

💡 Consejo WinHacks

Si una web no abre en tu PC pero sí funciona en tu celular, prueba primero ipconfig /flushdns antes de cambiar el navegador o reiniciar todo.

⚠️

Importante

Este comando no borra archivos personales, contraseñas ni historial del navegador. Solo limpia la caché DNS local de Windows.

Paso a paso

1

Abre el Símbolo del sistema

Presiona Win, escribe CMD y ejecútalo como administrador para evitar problemas de permisos.

2

Ejecuta el comando

Escribe el siguiente comando y presiona Enter.

ipconfig /flushdns
3

Comprueba el mensaje

Si todo salió correctamente aparecerá el mensaje indicando que la caché de resolución DNS fue vaciada correctamente.

4

Vuelve a abrir la página

Intenta acceder nuevamente al sitio web que presentaba problemas. Windows realizará una consulta DNS completamente nueva.

5

Renueva la dirección IP (opcional)

Si el problema continúa, puedes renovar también la dirección IP del equipo.

ipconfig /renew
6

Comprueba el DNS nuevamente

Usa nslookup para confirmar que Windows ya obtiene información nueva del servidor DNS.

nslookup google.com

¿Cuándo utilizar flushdns?

  • Una página dejó de abrir de repente.
  • Un dominio cambió de servidor.
  • Después de modificar los DNS.
  • Tras cambiar de hosting.
  • Errores de resolución de nombres.
  • Problemas de navegación después de usar VPN.
  • Diagnóstico de problemas de red.

✅ Resultado esperado

Windows eliminará todas las entradas DNS almacenadas localmente y realizará nuevas consultas al servidor DNS, solucionando muchos problemas relacionados con dominios y navegación.

Errores comunes

No se pudo vaciar la caché DNS

Abre el Símbolo del sistema como administrador e intenta ejecutar nuevamente el comando.

Las páginas siguen sin abrir

El problema puede estar en el servidor DNS, el router o tu proveedor de Internet. Prueba también con nslookup y ping para continuar el diagnóstico.

La conexión sigue siendo lenta

Limpiar la caché DNS no aumenta la velocidad de Internet. Solo elimina registros DNS almacenados localmente.

El comando terminó muy rápido

Es completamente normal. Windows únicamente elimina la caché DNS almacenada en memoria y finaliza el proceso en pocos segundos.

Preguntas frecuentes

¿ipconfig /flushdns elimina mi historial de navegación?

No. Solo elimina la caché DNS de Windows. No borra historial, cookies, contraseñas ni archivos del navegador.

¿Es seguro ejecutar flushdns?

Sí. Es un comando oficial de Windows utilizado por administradores y técnicos para solucionar problemas de resolución DNS.

¿Funciona en Windows 10 y Windows 11?

Sí. También está disponible en versiones anteriores de Windows compatibles con el comando ipconfig.

¿Cada cuánto debo usarlo?

Solo cuando tengas problemas relacionados con DNS o después de realizar cambios en la configuración de red. No es necesario ejecutarlo de forma periódica.
