Cómo ver claves WiFi guardadas en Windows con CMD

Si olvidaste la contraseña de una red WiFi que ya está guardada en tu PC, Windows puede mostrarla usando comandos oficiales desde CMD.

Mira cómo ver una clave WiFi guardada

En este Short verás cómo listar redes guardadas y mostrar la contraseña de una red WiFi desde CMD.

Qué aprenderás

  • Cómo ver redes WiFi guardadas en Windows.
  • Cómo recuperar la clave de una red autorizada.
  • Dónde aparece el campo “Contenido de la clave”.
  • Qué errores revisar si no aparece la contraseña.

Qué hace este comando

Windows guarda perfiles de redes WiFi a las que ya te conectaste. Con netsh wlan puedes listar esos perfiles y consultar detalles técnicos de una red específica.

Este método solo sirve para redes guardadas previamente en tu equipo. No sirve para hackear redes ni acceder a contraseñas de redes ajenas.

Consejo WinHacks

Copia primero el nombre exacto de la red desde la lista de perfiles. Si escribes mal una letra, el segundo comando puede fallar.

Advertencia de seguridad

Usa este comando solo en redes propias o donde tengas autorización. No compartas contraseñas WiFi sin permiso del dueño de la red.

Paso a paso para ver la clave WiFi

1

Abre CMD como administrador

Busca CMD en Windows, haz clic derecho y selecciona Ejecutar como administrador.

2

Lista las redes guardadas

Escribe este comando para ver los perfiles WiFi guardados:

netsh wlan show profiles
3

Copia el nombre de tu red

Busca el nombre de la red WiFi que quieres consultar. Debes escribirlo exactamente igual en el siguiente comando.

4

Muestra la información completa

Reemplaza NOMBREWIFI por el nombre real de tu red:

netsh wlan show profile "NOMBREWIFI" key=clear
5

Busca “Contenido de la clave”

En la salida del comando, baja hasta la sección de seguridad y busca el campo Contenido de la clave. Ahí aparece la contraseña guardada.

Ejemplo práctico

Si tu red se llama CasaWiFi, el comando quedaría así:

netsh wlan show profile "CasaWiFi" key=clear

Resultado esperado

Windows mostrará los detalles del perfil WiFi. Si tienes permisos suficientes y la clave está guardada, verás la contraseña junto a Contenido de la clave.

Errores comunes

No aparece “Contenido de la clave”

Puede ocurrir si el perfil no tiene una clave guardada, si no abriste CMD como administrador o si la red usa una configuración administrada por empresa.

El comando dice que el perfil no existe

Revisa el nombre exacto de la red. Si tiene espacios, debe ir entre comillas.

Veo muchos datos técnicos

Es normal. Solo necesitas buscar la sección de seguridad y el campo Contenido de la clave.

Preguntas frecuentes

¿Funciona en Windows 11?

Sí. También funciona en Windows 10 mientras el perfil WiFi esté guardado en el equipo.

¿Puedo ver claves de redes no guardadas?

No. Este método solo muestra redes guardadas previamente en tu PC y a las que tu usuario tiene acceso.

¿Es necesario instalar programas?

No. Este truco usa herramientas integradas de Windows desde CMD.
