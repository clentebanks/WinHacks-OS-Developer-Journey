¿Qué es hostname?

hostname es un comando integrado en Windows que muestra el nombre del equipo configurado en el sistema operativo. Este nombre identifica tu computadora dentro de una red local, Active Directory, servidores, scripts y herramientas de administración.

Aunque también puedes consultar este dato desde Configuración o Propiedades del sistema, hostname es mucho más rápido y suele utilizarse en automatización y soporte remoto.

💡 Consejo WinHacks

Si utilizas Escritorio Remoto, compartes archivos por red o administras varios equipos, conocer el nombre del equipo facilita muchísimo la identificación de cada computadora.

⚠️

Importante

El comando únicamente muestra el nombre configurado del equipo. No modifica ninguna configuración del sistema ni requiere permisos de administrador.

Paso a paso

1

Abre el Símbolo del sistema

Presiona Win, escribe CMD y abre el Símbolo del sistema. No necesitas permisos de administrador.

2

Ejecuta el comando

Escribe el siguiente comando y presiona Enter.

hostname
3

Observa el resultado

Windows mostrará inmediatamente el nombre asignado a tu computadora.

4

Utilízalo en PowerShell

El mismo comando funciona desde PowerShell.

hostname
5

Compara con Variables del Sistema

También puedes consultar el nombre utilizando una variable de entorno.

echo %COMPUTERNAME%
6

Consulta información adicional

Si necesitas más detalles sobre el equipo, utiliza el siguiente comando.

systeminfo

Errores comunes

El comando no se reconoce

Asegúrate de escribir exactamente hostname, sin espacios adicionales ni caracteres especiales.

El nombre mostrado no es el esperado

El comando muestra el nombre configurado actualmente en Windows. Si recientemente cambiaste el nombre del equipo, puede ser necesario reiniciar el sistema para que el cambio se aplique completamente.

El nombre es muy largo o extraño

Algunos fabricantes asignan nombres automáticos durante la instalación de Windows. Puedes cambiarlo desde la configuración del sistema si lo deseas.

Obtengo un nombre diferente al de la red

El nombre del equipo puede diferir del nombre visible en algunas aplicaciones o servicios. hostname siempre muestra el nombre registrado por Windows.

Preguntas frecuentes

¿hostname cambia el nombre del equipo?

No. Solo muestra el nombre actual del equipo. No modifica ninguna configuración del sistema.

¿Necesito permisos de administrador?

No. Cualquier usuario puede ejecutar hostname desde CMD o PowerShell.

¿Funciona en Windows 10 y Windows 11?

Sí. También está disponible en versiones anteriores de Windows y en Windows Server.

¿Cuál es la diferencia entre hostname y systeminfo?

hostname únicamente muestra el nombre del equipo, mientras que systeminfo presenta información completa del sistema operativo, hardware, memoria y red.
