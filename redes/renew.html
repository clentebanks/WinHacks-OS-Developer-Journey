ipconfig /renew: renueva la dirección IP de tu PC en Windows

El comando ipconfig /renew solicita una nueva dirección IP al router o servidor DHCP de tu red. Es una herramienta clave para recuperar conexión después de usar ipconfig /release, resolver conflictos de IP y reparar fallos de red.

¿Qué hace ipconfig /renew?

ipconfig /renew le pide a Windows que solicite una nueva dirección IP al servidor DHCP de la red. En una red doméstica, normalmente ese servidor DHCP es el router.

Este comando se usa cuando el equipo perdió conectividad, quedó sin IP después de un ipconfig /release, tiene conflicto de dirección o necesita actualizar la configuración recibida desde el router.

💡 Consejo WinHacks

Para una reparación rápida de red, la secuencia más usada es: ipconfig /release, luego ipconfig /renew y después ipconfig /flushdns.

⚠️

Importante

Si ejecutas ipconfig /renew y Windows no recibe una IP, puede existir un problema con el router, el adaptador de red, el cable Ethernet, la señal WiFi o el servicio DHCP.

Paso a paso

1

Abre el Símbolo del sistema

Presiona Win, escribe CMD y ejecútalo como administrador para evitar problemas de permisos.

2

Solicita una nueva dirección IP

Escribe el siguiente comando y presiona Enter.

ipconfig /renew
3

Espera la respuesta del servidor

Windows se comunicará con el servidor DHCP del router para obtener una nueva dirección IP automáticamente.

4

Comprueba la nueva dirección IP

Ejecuta el siguiente comando para verificar que Windows recibió una dirección IP válida.

ipconfig
5

Comprueba la conexión

Verifica que Internet funciona correctamente utilizando el comando ping.

ping google.com
6

Si continúa el problema

Limpia la caché DNS y vuelve a comprobar la resolución de nombres.

ipconfig /flushdns
nslookup google.com

¿Cuándo utilizar ipconfig /renew?

  • Después de ejecutar ipconfig /release.
  • Cuando el equipo perdió la dirección IP.
  • Errores del servidor DHCP.
  • Cambio de router o red local.
  • Problemas de conectividad después de reiniciar el router.
  • Conflictos de IP.
  • Diagnóstico de problemas de red.

✅ Resultado esperado

Windows obtendrá una nueva dirección IP desde el servidor DHCP, recuperando la conectividad normal de la red siempre que el servidor esté funcionando correctamente.

Errores comunes

No se obtiene una nueva dirección IP

Comprueba que el router tenga habilitado el servicio DHCP y que el adaptador de red esté conectado correctamente. Si es necesario, reinicia el router e intenta nuevamente.

Aparece "Media disconnected"

Este mensaje indica que Windows no detecta conexión física con la red. Revisa el cable Ethernet o vuelve a conectarte a la red Wi-Fi antes de ejecutar el comando.

El comando tarda demasiado

Puede deberse a que Windows está intentando contactar al servidor DHCP y no recibe respuesta. Comprueba que el router esté funcionando correctamente.

La IP sigue siendo la misma

Es completamente normal. El servidor DHCP puede volver a asignar la misma dirección IP si continúa disponible.

Preguntas frecuentes

¿ipconfig /renew cambia mi dirección IP pública?

No. Solo renueva la dirección IP privada asignada por el router. La dirección IP pública depende de tu proveedor de Internet.

¿Funciona con conexiones Wi-Fi y Ethernet?

Sí. El comando funciona con cualquier adaptador configurado para obtener una dirección IP automáticamente mediante DHCP.

¿Necesito ejecutar primero ipconfig /release?

No siempre. ipconfig /renew puede ejecutarse por sí solo, aunque normalmente se utiliza después de ipconfig /release para renovar completamente la configuración IP.

¿Es seguro usar este comando?

Sí. Es un comando oficial de Microsoft utilizado por administradores, técnicos y usuarios para solucionar problemas de conectividad.
