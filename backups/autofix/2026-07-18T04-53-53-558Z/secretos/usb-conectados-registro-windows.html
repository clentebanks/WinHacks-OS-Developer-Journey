Windows recuerda dispositivos USB conectados al sistema

Windows puede guardar registros internos de dispositivos USB conectados, incluso si ya no están conectados al equipo. Uno de los lugares más conocidos para revisar esta información es la clave USBSTOR dentro del Editor del Registro.

Mira dónde Windows guarda registros USB

¿Qué es USBSTOR?

USBSTOR es una clave del Registro de Windows asociada a dispositivos de almacenamiento USB. Ahí pueden aparecer entradas relacionadas con memorias USB, discos externos, lectores de tarjetas u otros dispositivos de almacenamiento conectados anteriormente.

Esta información puede ser útil para diagnóstico, auditoría básica, privacidad digital o para entender cómo Windows maneja dispositivos conectados al sistema.

💡 Consejo WinHacks

Usa esta revisión solo en tu propio equipo o en sistemas donde tienes autorización. Estos registros pueden contener información sensible sobre dispositivos conectados.

🚨

No borres claves al azar

El Registro de Windows es delicado. Puedes revisar la información, pero no elimines claves ni valores si no sabes exactamente qué efecto tendrá.

Paso a paso

1

Abre Ejecutar

Presiona Win + R para abrir la ventana Ejecutar.

2

Abre el Editor del Registro

Escribe este comando y presiona Enter:

regedit
3

Ve a la clave USBSTOR

En la barra superior del Registro, pega esta ruta:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\USBSTOR
4

Revisa las carpetas detectadas

Dentro de USBSTOR pueden aparecer carpetas con nombres relacionados con fabricantes, modelos o tipos de dispositivos USB conectados.

5

Abre una entrada USB

Al expandir una entrada, puedes ver subclaves internas asociadas al dispositivo. Algunas pueden incluir identificadores o información técnica del hardware.

6

Revisa sin modificar

Observa la información, pero evita cambiar permisos, eliminar claves o modificar valores. Para diagnóstico básico, solo necesitas revisar.

Qué información podrías encontrar

Nombre o tipo del dispositivo

Algunas entradas pueden incluir pistas sobre la marca, modelo o tipo de almacenamiento USB.

Identificadores internos

Windows puede guardar identificadores técnicos que ayudan a reconocer un dispositivo cuando vuelve a conectarse.

Entradas antiguas

Es posible encontrar registros de dispositivos que ya no están conectados actualmente.

✅ Resultado esperado

Al abrir USBSTOR, podrás ver entradas internas asociadas a dispositivos de almacenamiento USB que Windows ha detectado en el equipo.

Errores comunes

No encuentro USBSTOR

Verifica que pegaste la ruta completa: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\USBSTOR.

Windows no me deja editar

Es normal. Esta zona del Registro puede estar protegida. Para este tutorial no necesitas editar nada.

No aparece el USB que esperaba

Puede depender del tipo de dispositivo, permisos, historial del sistema o si Windows lo registró como almacenamiento USB.

Preguntas frecuentes

¿Esto muestra todos los USB conectados?

No necesariamente. USBSTOR se enfoca principalmente en dispositivos de almacenamiento USB. Otros periféricos pueden quedar en claves diferentes.

¿Funciona en Windows 11?

Sí. También puede revisarse en Windows 10.

¿Puedo borrar estos rastros?

Técnicamente existen formas avanzadas, pero no es recomendable borrar claves del Registro sin saber exactamente qué haces. Puedes afectar reconocimiento de dispositivos.

¿Necesito permisos de administrador?

Para abrir y revisar algunas zonas de HKEY_LOCAL_MACHINE, Windows puede pedir permisos elevados o confirmación de UAC.
