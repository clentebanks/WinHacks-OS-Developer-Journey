Windows registra comandos que ejecutas con Win + R

Windows puede guardar un historial de comandos, rutas y accesos usados desde la ventana Ejecutar. Ese registro se encuentra en una clave del Registro llamada RunMRU, y la mayoría de usuarios nunca la revisa.

¿Qué es RunMRU?

RunMRU es una clave del Registro de Windows donde pueden quedar guardados comandos recientes usados desde la ventana Ejecutar, que se abre con Win + R.

Ahí pueden aparecer comandos como cmd, regedit, msconfig, ncpa.cpl, rutas de carpetas o accesos usados recientemente por el usuario actual.

💡 Consejo WinHacks

Esta clave puede servir para recordar comandos que ejecutaste antes o para revisar actividad reciente en tu propio usuario de Windows.

🚨

No edites valores al azar

El Registro de Windows es delicado. Puedes revisar esta clave sin problema, pero no elimines ni modifiques entradas si no sabes exactamente qué estás haciendo.

Paso a paso

1

Abre Ejecutar

Presiona Win + R para abrir la ventana Ejecutar de Windows.

2

Abre el Editor del Registro

Escribe este comando y presiona Enter:

regedit
3

Ve a la clave RunMRU

En la barra superior del Editor del Registro, pega esta ruta:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\RunMRU
4

Revisa los valores guardados

En el panel derecho verás valores con letras como a, b, c o similares. Cada uno puede representar un comando ejecutado.

5

Identifica el orden con MRUList

El valor MRUList indica el orden de uso reciente. Si contiene letras como cab, significa que Windows ordenó esos comandos según uso reciente.

Ejemplos de comandos que podrías encontrar

Herramientas del sistema

Puedes encontrar comandos como cmd, powershell, regedit, msconfig o services.msc.

Paneles de configuración

También pueden aparecer accesos como ncpa.cpl, appwiz.cpl, control o sysdm.cpl.

Rutas o carpetas

Si ejecutaste una ruta desde Win + R, también puede quedar registrada en esta ubicación.

✅ Resultado esperado

Al abrir la clave RunMRU, podrás ver comandos, rutas o accesos usados recientemente desde la ventana Ejecutar del usuario actual.

Errores comunes

No encuentro RunMRU

Verifica que la ruta esté escrita correctamente: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\RunMRU.

La clave aparece vacía

Puede ocurrir si no has usado la ventana Ejecutar recientemente o si el historial fue limpiado.

Veo muchas claves parecidas

No entres en carpetas al azar. La ruta correcta está dentro de CurrentVersion\Explorer\RunMRU, no en claves como GameBar, IME o Input.

Preguntas frecuentes

¿Esto muestra todo lo que hago en Windows?

No. Solo puede mostrar comandos o rutas usados desde la ventana Ejecutar. No es un historial completo de actividad del equipo.

¿Funciona en Windows 11?

Sí. También funciona en Windows 10.

¿Puedo borrar este historial?

Sí, pero no lo hagas directamente desde el Registro si no tienes experiencia. Es mejor borrar el historial de Ejecutar desde opciones de privacidad o limpiar entradas con cuidado.

¿Necesito permisos de administrador?

Para revisar HKEY_CURRENT_USER, normalmente no necesitas permisos elevados. Aun así, Windows puede pedir confirmación para abrir el Editor del Registro.
