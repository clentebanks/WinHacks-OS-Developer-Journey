Qué es este menú secreto de Windows

El menú contextual es el menú que aparece cuando haces clic derecho en el escritorio, una carpeta o un archivo. Windows permite extenderlo para agregar accesos directos personalizados.

Este truco se puede usar para abrir herramientas como CMD, PowerShell, el Administrador de tareas, Configuración, Bloc de notas o aplicaciones instaladas. No mejora el rendimiento por sí solo, pero acelera mucho el flujo de trabajo.

Consejo WinHacks

Agrega solo herramientas que uses todos los días. Si colocas demasiadas opciones, el menú se vuelve lento, desordenado y menos útil.

Advertencia importante

Este ajuste puede tocar el Registro de Windows. Antes de modificarlo, crea un punto de restauración o exporta la clave que vas a editar.

Paso a paso para crear un acceso en el clic derecho

1

Abre el Editor del Registro

Presiona Win + R, escribe regedit y presiona Enter.

regedit
2

Ve a la clave del escritorio

Para agregar una opción al clic derecho del escritorio, navega a esta ruta:

HKEY_CLASSES_ROOT\Directory\Background\shell
3

Crea una nueva clave

Haz clic derecho sobre shell, selecciona Nuevo > Clave y ponle un nombre, por ejemplo:

WinHacks Tools
4

Crea la clave command

Dentro de la clave nueva, crea otra clave llamada exactamente:

command
5

Agrega el comando o aplicación

En el valor predeterminado de command, escribe la ruta o comando que quieres abrir. Por ejemplo, para abrir CMD:

cmd.exe

Ejemplos útiles para tu menú personalizado

  • cmd.exe para abrir CMD.
  • powershell.exe para abrir PowerShell.
  • taskmgr.exe para abrir el Administrador de tareas.
  • control.exe para abrir el Panel de control.
  • notepad.exe para abrir Bloc de notas.

Archivo .reg de ejemplo

También puedes crear un archivo .reg para agregar una opción de CMD al clic derecho del escritorio:

Windows Registry Editor Version 5.00 [HKEY_CLASSES_ROOT\Directory\Background\shell\WinHacks CMD] @="Abrir CMD WinHacks" [HKEY_CLASSES_ROOT\Directory\Background\shell\WinHacks CMD\command] @="cmd.exe"

Resultado esperado

Al hacer clic derecho en el escritorio, aparecerá tu nueva opción personalizada. Desde ahí podrás abrir herramientas, comandos o aplicaciones sin buscarlas manualmente.

Errores comunes

No aparece la opción en el clic derecho

Revisa que la clave esté dentro de HKEY_CLASSES_ROOT\Directory\Background\shell y que exista una subclave llamada command.

La opción aparece, pero no abre nada

Verifica que el valor dentro de command tenga una ruta válida o un ejecutable correcto como cmd.exe o taskmgr.exe.

Quiero eliminar el menú personalizado

Vuelve a la misma ruta del Registro y elimina la clave que creaste, por ejemplo WinHacks CMD o WinHacks Tools.

Preguntas frecuentes

¿Esto funciona en Windows 11?

Sí, aunque en Windows 11 algunas opciones pueden aparecer detrás de Mostrar más opciones según la configuración del menú contextual.

¿Es peligroso editar el Registro?

Puede serlo si modificas claves incorrectas. En este caso solo debes crear una clave nueva en una ruta específica. Aun así, crea una copia antes de cambiar algo.

¿Puedo agregar varias herramientas?

Sí. Puedes crear varias claves dentro de shell, una para cada herramienta que quieras abrir desde el clic derecho.
