Cambia el nombre de un disco en Windows

Aprende a cambiar el nombre visible de un disco, unidad USB o partición en Windows 10 y Windows 11 para identificar tus unidades de forma más clara.

Qué aprenderás

  • Cómo cambiar el nombre de un disco desde el Explorador.
  • Cómo renombrar una unidad USB.
  • Cómo cambiar la etiqueta desde Propiedades.
  • Cómo hacerlo con CMD usando label.
  • Qué errores evitar al renombrar unidades.

¿Qué hace este truco?

Cambiar el nombre de un disco solo modifica la etiqueta visible de la unidad. Por ejemplo, puedes cambiar una unidad llamada Disco local por nombres como Trabajo, Backups, WinHacks USB o Juegos.

Este cambio no borra archivos, no formatea la unidad y no cambia la letra del disco. Solo ayuda a identificar mejor cada unidad dentro del Explorador de archivos.

💡 Consejo WinHacks

Usa nombres cortos y claros. Por ejemplo: Backup, Trabajo, USB Tools o Juegos.

⚠️

Antes de continuar

No confundas cambiar el nombre del disco con cambiar la letra de unidad. Renombrar no cambia rutas como C:, D: o E:.

Paso a paso

1

Abre el Explorador de archivos

Presiona Win + E o abre el Explorador de archivos desde la barra de tareas.

Captura pendiente: Explorador de archivos abierto en Este equipo.
2

Entra en Este equipo

En el panel izquierdo, selecciona Este equipo para ver tus discos, particiones y unidades USB conectadas.

Captura pendiente: sección Este equipo con discos visibles.
3

Haz clic derecho sobre la unidad

Selecciona el disco, partición o USB que quieres renombrar. Haz clic derecho y elige Cambiar nombre.

Captura pendiente: menú contextual con Cambiar nombre resaltado.
4

Escribe el nuevo nombre

Coloca un nombre corto, fácil de reconocer y presiona Enter. Por ejemplo:

WinHacks USB
Captura pendiente: unidad renombrada como WinHacks USB.
5

Confirma permisos si Windows lo solicita

En algunas unidades, Windows puede pedir permisos de administrador. Confirma el cambio si la unidad es tuya y tienes autorización para modificarla.

Captura pendiente: aviso de permisos de administrador.

Método alternativo desde Propiedades

También puedes cambiar el nombre desde las propiedades de la unidad.

1

Abre Propiedades de la unidad

Haz clic derecho sobre el disco o USB y selecciona Propiedades.

Captura pendiente: opción Propiedades en la unidad.
2

Cambia la etiqueta del disco

En la parte superior de la ventana, escribe el nuevo nombre de la unidad y presiona Aplicar.

Captura pendiente: campo de nombre dentro de Propiedades.

Comando para cambiar el nombre desde CMD

Usar el comando label

También puedes cambiar el nombre de una unidad desde CMD usando label. Sustituye la letra y el nombre según tu caso.

label E: WinHacks USB

Ejemplo para una unidad D:

Este ejemplo cambia la etiqueta de la unidad D: a Backups.

label D: Backups
⚠️

Importante sobre CMD

Antes de usar label, verifica bien la letra de unidad. No uses una letra al azar, especialmente si tienes varios discos conectados.

Qué debes revisar primero

Antes de renombrar una unidad, confirma que estás seleccionando el disco correcto. Revisa la letra de unidad, el tamaño del disco y el contenido visible.

Si es una USB, puedes desconectar otras unidades externas para evitar confusiones.

✅ Resultado esperado

Al terminar, el disco, partición o USB aparecerá con el nuevo nombre dentro del Explorador de archivos, sin borrar tus archivos ni cambiar la letra de unidad.

Cuándo usar este truco

  • Cuando tienes varias unidades conectadas.
  • Cuando quieres identificar rápido una USB.
  • Cuando separas discos para juegos, trabajo o backups.
  • Cuando preparas una unidad técnica de herramientas.
  • Cuando quieres organizar mejor el Explorador de archivos.

Errores comunes

Confundir nombre con letra de unidad

Cambiar el nombre no cambia la letra. Una unidad puede llamarse Backups y seguir siendo D:.

Renombrar la unidad equivocada

Antes de confirmar, revisa la letra, el tamaño y el contenido del disco para evitar modificar la unidad incorrecta.

Usar nombres demasiado largos

Aunque Windows permite nombres descriptivos, es mejor usar etiquetas cortas para que se vean bien en el Explorador.

Preguntas frecuentes

¿Cambiar el nombre del disco borra archivos?

No. Solo cambia la etiqueta visible de la unidad. Tus archivos permanecen igual.

¿Esto cambia la letra C:, D: o E:?

No. Cambiar el nombre no modifica la letra de unidad. Para eso se usa Administración de discos.

¿Puedo renombrar una USB?

Sí. Puedes renombrar memorias USB, discos externos y particiones internas.

¿Necesito permisos de administrador?

Depende de la unidad. En discos internos o protegidos, Windows puede pedir permisos. En una USB propia normalmente es más simple.

¿Puedo hacerlo desde CMD?

Sí. Puedes usar el comando label seguido de la letra de unidad y el nuevo nombre.
