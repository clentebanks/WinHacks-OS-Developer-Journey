Cómo quitar las flechas de los accesos directos en Windows

Aprende a eliminar las flechas pequeñas que aparecen en los accesos directos del escritorio usando el Registro de Windows, sin instalar programas externos.

Qué aprenderás

  • Qué son las flechas de los accesos directos.
  • Cómo quitarlas desde el Registro de Windows.
  • Cómo restaurarlas si quieres volver al estado original.

¿Qué son esas flechas?

Windows muestra una pequeña flecha sobre los accesos directos para diferenciarlos de archivos, carpetas o programas reales.

El acceso directo no es el programa original. Solo es un enlace que apunta hacia otra ubicación. Por eso Windows coloca esa flecha como indicador visual.

💡 Consejo WinHacks

Este cambio es visual. No elimina tus accesos directos, no borra programas y no cambia la ubicación real de tus archivos.

⚠️

Antes de continuar

Este tutorial usa el Editor del Registro. Antes de aplicar cambios, crea un punto de restauración de Windows por seguridad.

Paso a paso

1

Crea una carpeta para el icono vacío

Primero crea una carpeta fija donde guardarás el icono vacío. No muevas esta carpeta después, porque Windows usará esa ruta para ocultar la flecha.

C:\Iconos
Captura pendiente: carpeta C:\Iconos creada.
2

Guarda el archivo blank.ico

Dentro de esa carpeta coloca un icono vacío llamado blank.ico. Ese archivo será usado para reemplazar la flecha visual de los accesos directos.

C:\Iconos\blank.ico

⚡ Truco WinHacks

Usa siempre una ruta simple como C:\Iconos\blank.ico. Evita guardarlo en Descargas, Escritorio o carpetas que puedas borrar después.

3

Abre el Editor del Registro

Presiona Win + R, escribe regedit y presiona Enter. Acepta el permiso de administrador si Windows lo solicita.

regedit
4

Ve hasta la carpeta Explorer

En el Editor del Registro, navega hasta esta ruta. Aquí es donde se controla parte del comportamiento visual del Explorador de Windows.

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer
Captura pendiente: ruta Explorer dentro del Registro.
5

Crea la clave Shell Icons

Dentro de la carpeta Explorer, haz clic derecho, selecciona Nuevo > Clave y nómbrala exactamente:

Shell Icons

Importante

En muchos equipos esta clave no existe. Si no la encuentras, debes crearla manualmente. Eso es normal.

6

Crea el valor 29

Entra a Shell Icons. Luego haz clic derecho en el panel derecho y crea un Nuevo valor de cadena. El nombre debe ser:

29

Después haz doble clic sobre ese valor y pega la ruta del icono vacío.

C:\Iconos\blank.ico
7

Reinicia el Explorador de Windows

Para ver el cambio, abre el Administrador de tareas, busca Explorador de Windows, haz clic derecho y selecciona Reiniciar.

Captura pendiente: reiniciar Explorador de Windows desde el Administrador de tareas.

✅ Resultado esperado

Después de reiniciar el Explorador, las flechas pequeñas de los accesos directos deberían desaparecer y tu escritorio se verá más limpio.

Cómo volver atrás

Si quieres restaurar las flechas originales, vuelve al Registro y elimina el valor 29 dentro de Shell Icons.

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Shell Icons

Luego reinicia el Explorador de Windows o reinicia tu computadora.

Errores comunes

Las flechas no desaparecen

Verifica que el archivo blank.ico exista realmente en C:\Iconos y que la ruta esté escrita correctamente en el valor 29.

No encuentro Shell Icons

Es normal. Si no existe, debes crear la clave manualmente dentro de Explorer.

El cambio se ve raro

Puede ser caché de iconos. Reinicia el Explorador de Windows o reinicia la PC.

Preguntas frecuentes

¿Funciona en Windows 11?

Sí. Este método funciona en Windows 11 si se crea correctamente la clave Shell Icons y el valor 29.

¿Funciona en Windows 10?

Sí. También puede funcionar en Windows 10 usando la misma ruta del Registro.

¿Se dañan mis accesos directos?

No. Solo se modifica el icono superpuesto. Los accesos directos siguen funcionando igual.

¿Necesito instalar programas?

No. Solo necesitas el Editor del Registro y un archivo de icono vacío.

¿Puedo deshacer el cambio?

Sí. Elimina el valor 29 y reinicia el Explorador de Windows.
