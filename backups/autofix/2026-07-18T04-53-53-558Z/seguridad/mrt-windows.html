¿Qué es MRT?

MRT significa Microsoft Windows Malicious Software Removal Tool. Es una herramienta integrada de Microsoft que ayuda a detectar y eliminar ciertas familias de software malintencionado conocidas.

No reemplaza a Windows Defender ni a un antivirus completo, pero puede servir como revisión adicional cuando sospechas que algo extraño ocurre en el sistema.

Antes de continuar

MRT no detecta todo tipo de malware. Úsalo como herramienta adicional, no como reemplazo de Windows Defender o de una solución de seguridad activa.

Tipos de análisis en MRT

Análisis rápido

Revisa las zonas más comunes donde puede esconderse software malintencionado. Es la opción más rápida para una revisión básica.

Análisis completo

Revisa todo el equipo. Puede tardar bastante más, pero es mejor si sospechas una infección o comportamiento extraño.

Análisis personalizado

Permite revisar una carpeta o ubicación específica que tú selecciones.

Qué hacer si MRT detecta una amenaza

  • Permite que MRT complete el proceso de eliminación.
  • Reinicia Windows si la herramienta lo solicita.
  • Ejecuta un análisis completo con Windows Defender.
  • Cambia contraseñas importantes si sospechas robo de información.
  • Actualiza Windows y tus navegadores.
  • Evita volver a abrir archivos sospechosos.

Cuándo usar MRT

  • Cuando tu PC tiene comportamiento extraño.
  • Cuando aparecen ventanas o procesos desconocidos.
  • Cuando el navegador abre páginas raras.
  • Después de descargar archivos sospechosos.
  • Como revisión adicional junto a Windows Defender.

✅ Resultado esperado

Al finalizar, MRT mostrará si encontró software malintencionado conocido y si pudo eliminarlo. Si no encuentra amenazas, úsalo como una señal adicional, no como garantía absoluta de que el sistema está limpio.

Errores comunes

Pensar que MRT reemplaza al antivirus

MRT es una herramienta de eliminación puntual. No ofrece protección en tiempo real como Windows Defender.

No actualizar Windows

Mantener Windows actualizado ayuda a recibir mejoras de seguridad y definiciones más recientes para las herramientas integradas.

Ignorar señales después de un análisis limpio

Si MRT no detecta nada pero el equipo sigue raro, ejecuta Windows Defender, revisa programas instalados y verifica extensiones del navegador.

Preguntas frecuentes

¿MRT viene instalado en Windows?

Sí. En muchas instalaciones de Windows puedes abrirlo directamente con Win + R y el comando mrt.

¿MRT es un antivirus?

No. Es una herramienta de eliminación de malware específico. Para protección en tiempo real usa Windows Defender u otra solución confiable.

¿Necesito Internet para usar MRT?

Puedes abrir la herramienta sin Internet, pero conviene mantener Windows actualizado para tener revisiones más recientes.

¿El análisis completo tarda mucho?

Sí, puede tardar bastante dependiendo del tamaño del disco, la cantidad de archivos y el rendimiento del equipo.

¿Qué hago si MRT encuentra malware?

Deja que complete la eliminación, reinicia si lo pide y luego ejecuta un análisis completo desde Seguridad de Windows.
