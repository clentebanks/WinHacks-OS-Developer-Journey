¿Qué es el Historial de protección?

El Historial de protección es una sección de Seguridad de Windows donde Microsoft Defender registra todas las amenazas detectadas, archivos bloqueados, elementos en cuarentena y acciones realizadas automáticamente.

Si alguna vez descargaste un archivo sospechoso, aquí podrás comprobar si Windows actuó para proteger el sistema.

💡 Consejo WinHacks

Revisa este historial de vez en cuando. Muchas personas no saben que Windows ya bloqueó amenazas sin mostrar una notificación evidente.

Importante

No restaures archivos desde la cuarentena a menos que estés completamente seguro de que se trata de un falso positivo.

1

Abre Seguridad de Windows

Presiona Win + R y ejecuta:

windowsdefender:

2

Selecciona la sección donde Windows Defender administra las amenazas detectadas.

3

Haz clic en Historial de protección

Se mostrará una lista con todas las detecciones recientes.

Captura pendiente: Historial de protección.

Consulta los detalles

Haz clic sobre cualquier detección para conocer el archivo, la ubicación, la gravedad y la acción realizada por Windows.

Realiza un análisis completo

Estados más comunes

Amenaza bloqueada

Windows detectó el archivo y evitó su ejecución.

Elemento en cuarentena

El archivo fue aislado para impedir que afecte el sistema.

Acción completada

La amenaza fue eliminada o neutralizada correctamente.

Acción requerida

Necesitas decidir qué hacer con la amenaza detectada.

Comandos útiles

ms-settings:windowsdefender

mrt

Qué revisar

  • Archivos puestos en cuarentena.
  • Amenazas repetidas.
  • Acciones pendientes.
  • Elementos restaurados.
  • Fechas de detección.
  • Ubicación de los archivos afectados.

✅ Resultado esperado

Podrás saber si Windows Defender detectó malware, qué archivos fueron afectados y qué acciones tomó automáticamente.

Cuándo revisar el historial

  • Después de descargar programas.
  • Después de conectar una memoria USB.
  • Cuando Windows muestra alertas.
  • Si el equipo presenta comportamientos extraños.
  • Después de ejecutar un análisis completo.

Errores comunes

Ignorar las advertencias

No asumas que todas las amenazas son falsas. Revisa la información antes de restaurar archivos.

Restaurar archivos peligrosos

Solo restaura un archivo si estás completamente seguro de que es legítimo.

No ejecutar un análisis adicional

Si aparecen amenazas repetidas, realiza un análisis completo o sin conexión para mayor seguridad.

Preguntas frecuentes

¿Puedo eliminar el historial?

Windows elimina automáticamente registros antiguos con el tiempo. No es recomendable borrarlos manualmente.

¿Qué significa cuarentena?

El archivo queda aislado para impedir que pueda ejecutarse o afectar Windows.

¿Debo restaurar un archivo?

Solo si sabes con certeza que es un falso positivo.

¿Qué hago si aparecen muchas amenazas?

Ejecuta un análisis completo, un análisis sin conexión y revisa los programas instalados recientemente.
