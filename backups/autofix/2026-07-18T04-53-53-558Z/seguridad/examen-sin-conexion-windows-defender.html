El escaneo secreto de Windows que elimina malware antes de iniciar el sistema

Windows incluye un modo avanzado llamado Examen sin conexión. Este análisis reinicia tu PC y escanea el sistema antes de que Windows cargue por completo, lo que puede ayudar contra amenazas difíciles de detectar en una sesión normal.

¿Qué es el Examen sin conexión?

El Examen sin conexión es un análisis avanzado de Microsoft Defender. A diferencia de un examen rápido normal, este modo reinicia el equipo y analiza el sistema en un entorno separado antes de cargar Windows completamente.

Esto puede ayudar cuando sospechas de malware persistente, procesos extraños, ventanas raras, redirecciones del navegador o comportamiento anormal que no desaparece con un análisis rápido.

💡 Consejo WinHacks

Antes de usar este modo, ejecuta primero un Examen rápido. Si Windows sigue raro o detectas señales sospechosas, entonces pasa al Examen sin conexión.

🚨

Antes de reiniciar

Guarda todos tus archivos abiertos. Este examen reinicia la PC y puede tardar varios minutos. No lo hagas mientras estés editando, descargando archivos importantes o en una reunión.

Paso a paso

1

Abre Seguridad de Windows

Presiona Win, busca Seguridad de Windows y abre la aplicación oficial del sistema.

2

Entra en Protección antivirus

Haz clic en Protección antivirus y contra amenazas. Esta sección controla los análisis de Microsoft Defender.

3

Abre Opciones de examen

Busca y selecciona Opciones de examen. Ahí verás los modos de análisis disponibles.

4

Selecciona Examen sin conexión

Marca la opción Examen sin conexión de Microsoft Defender. Este modo reiniciará el equipo para escanear antes de cargar Windows.

5

Inicia el examen

Presiona Examinar ahora. Windows te avisará que el equipo debe reiniciarse. Guarda todo antes de continuar.

6

Espera el reinicio y el análisis

La PC se reiniciará y Microsoft Defender ejecutará el examen sin conexión. Cuando termine, Windows volverá a iniciar normalmente.

Cuándo conviene usar este escaneo

Cuando el análisis normal no basta

Si el examen rápido no encuentra nada, pero tu PC sigue comportándose raro, este modo puede ser el siguiente paso.

Cuando hay procesos sospechosos

Si ves procesos extraños, alto consumo sin explicación o conexiones sospechosas, combina este examen con una revisión básica del Administrador de tareas.

Después de abrir un archivo dudoso

Si ejecutaste un archivo sospechoso o descargaste algo de una fuente no confiable, este examen puede ayudarte a revisar el sistema más a fondo.

✅ Resultado esperado

Windows reiniciará, ejecutará Microsoft Defender en modo sin conexión y luego volverá al escritorio. Después podrás revisar el historial de protección para ver si detectó amenazas.

Errores comunes

No encuentro Opciones de examen

Entra en Seguridad de Windows, luego en Protección antivirus y contra amenazas. La opción puede aparecer más abajo dependiendo del tamaño de la ventana.

La PC se reinició de repente

Es normal. El Examen sin conexión necesita reiniciar para ejecutarse antes de que Windows cargue completamente.

No veo resultados al volver

Abre Seguridad de Windows y revisa el Historial de protección para ver acciones recientes.

Preguntas frecuentes

¿Este examen elimina todos los virus?

No existe un método que garantice eliminar todo en todos los casos, pero este análisis es más profundo que un examen rápido normal y puede detectar amenazas difíciles de ocultar.

¿Pierdo mis archivos?

No debería eliminar archivos personales normales. Si detecta malware, puede ponerlo en cuarentena o eliminar archivos infectados.

¿Necesito Internet?

Conviene tener Windows actualizado antes de ejecutarlo para que Defender tenga definiciones recientes.

¿Funciona en Windows 11?

Sí. También está disponible en Windows 10.
