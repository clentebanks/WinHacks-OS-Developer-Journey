Redes Windows

El comando secreto para ver dispositivos conectados a tu red

Windows incluye una herramienta de red llamada ARP. Con el comando arp -a puedes ver direcciones IP y direcciones MAC que tu PC ha detectado dentro de tu red local.

Qué aprenderás

  • Qué significan las direcciones IP y MAC que aparecen.
  • Qué limitaciones tiene este comando.
  • Cómo usarlo como primer diagnóstico de tu red local.

¿Qué hace el comando arp -a?

El comando arp -a muestra la tabla ARP de tu equipo. Esta tabla relaciona direcciones IP locales con direcciones físicas o MAC que Windows ha detectado en la red.

Es importante entender que arp -a no muestra necesariamente todos los dispositivos conectados al WiFi. Muestra los dispositivos con los que tu PC ha tenido comunicación reciente o que aparecen en su caché ARP.

💡 Consejo WinHacks

Para que aparezcan más dispositivos, puedes abrir el navegador, entrar a tu router o hacer ping a tu red local antes de ejecutar arp -a.

🚨

Advertencia importante

Este comando no reemplaza el panel del router. Si quieres confirmar quién está conectado realmente a tu WiFi, revisa también la lista de clientes conectados dentro de la configuración del router.

Paso a paso

1

Abre CMD

Presiona Win, escribe CMD y abre el Símbolo del sistema.

2

Ejecuta el comando arp

Escribe este comando y presiona Enter:

arp -a
3

Revisa las direcciones IP

En la columna de Dirección de Internet verás IPs locales detectadas, normalmente con formatos como 192.168.1.x o 192.168.0.x.

4

Revisa las direcciones MAC

En la columna de Dirección física verás direcciones MAC asociadas a algunas IPs. Estas ayudan a diferenciar dispositivos dentro de la red.

5

Compara con tu router

Entra al panel de tu router y compara la lista de dispositivos conectados con las IPs que viste en CMD.

Cómo interpretar la salida

Dirección de Internet

Es la dirección IP local detectada por Windows. Puede pertenecer a tu router, tu teléfono, una laptop, una impresora, una TV o cualquier equipo de la red.

Dirección física

Es la dirección MAC asociada a esa IP. Sirve para identificar el hardware de red, aunque algunos dispositivos modernos pueden usar MAC aleatorias por privacidad.

Tipo

Puede aparecer como dinámico o estático. Lo más común en redes domésticas es ver entradas dinámicas.

✅ Resultado esperado

Al ejecutar arp -a, Windows mostrará IPs y direcciones físicas detectadas en la red local. Esto te da una primera pista sobre actividad de dispositivos cercanos o conectados recientemente.

Errores comunes

No veo todos los dispositivos

Es normal. arp -a solo muestra entradas que tu PC conoce en ese momento. Para ver todos los conectados, revisa el panel del router.

No sé cuál dispositivo es cuál

Compara las IPs con la lista de clientes del router. También puedes desconectar un dispositivo y volver a revisar para identificarlo.

Veo direcciones raras

Algunas entradas pueden ser del router, multicast, broadcast o servicios de red. No todas representan un intruso.

Preguntas frecuentes

¿arp -a muestra quién está usando mi WiFi?

Puede mostrar dispositivos detectados por tu PC, pero no es una lista completa y definitiva de todos los conectados. Para confirmar, revisa el router.

¿Funciona en Windows 11?

Sí. También funciona en Windows 10 desde CMD o Terminal.

¿Necesito abrir CMD como administrador?

Normalmente no. El comando arp -a suele funcionar desde CMD normal.

¿Puedo bloquear dispositivos desde arp?

No. Para bloquear dispositivos debes entrar al panel del router y cambiar contraseña, usar lista de control de acceso o administrar clientes conectados.
