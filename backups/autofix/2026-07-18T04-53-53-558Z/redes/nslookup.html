nslookup: el comando para revisar DNS desde Windows

Windows incluye el comando nslookup, una herramienta de diagnóstico que permite consultar registros DNS, resolver dominios, comprobar direcciones IP y detectar problemas de conexión relacionados con servidores DNS.

¿Qué es nslookup?

nslookup significa Name Server Lookup. Es una herramienta integrada en Windows que permite consultar servidores DNS para obtener información sobre nombres de dominio y direcciones IP.

Cuando escribes una página web en el navegador, Windows necesita resolver ese dominio a una dirección IP. Si esa resolución falla, puedes tener problemas para abrir sitios aunque tu conexión a Internet parezca activa.

💡 Consejo WinHacks

Si una página no carga pero otras sí funcionan, prueba primero con nslookup. Te ayuda a saber si el problema está en el DNS, en el dominio o en tu conexión.

⚠️

Importante

nslookup no modifica tu red ni cambia configuraciones. Solo consulta información DNS, por lo que es seguro usarlo para diagnóstico.

Paso a paso

1

Abre CMD

Presiona Win, escribe CMD y abre el Símbolo del sistema. No necesitas permisos de administrador para consultas DNS básicas.

2

Consulta un dominio

Escribe nslookup seguido del dominio que quieres revisar.

nslookup google.com
3

Revisa el servidor DNS usado

La primera parte del resultado muestra el servidor DNS que respondió la consulta. Puede ser tu router, tu proveedor de Internet o un DNS público.

4

Revisa la dirección IP del dominio

En la salida verás una o varias direcciones IP asociadas al dominio. Esto confirma que el DNS pudo resolver el nombre correctamente.

5

Prueba con un DNS específico

También puedes consultar usando un servidor DNS concreto, como Google DNS o Cloudflare DNS.

nslookup google.com 8.8.8.8
nslookup google.com 1.1.1.1
6

Consulta registros MX

Si quieres revisar servidores de correo de un dominio, puedes consultar registros MX:

nslookup -type=mx dominio.com

Comandos útiles con nslookup

Consulta básica

nslookup ejemplo.com

Usar DNS de Google

nslookup ejemplo.com 8.8.8.8

Usar DNS de Cloudflare

nslookup ejemplo.com 1.1.1.1

Consultar registros MX

nslookup -type=mx ejemplo.com

¿Cuándo usar nslookup?

  • Una página no carga, pero otras sí funcionan.
  • Quieres saber qué IP responde un dominio.
  • Sospechas problemas con tu DNS.
  • Quieres comparar DNS de tu proveedor contra DNS públicos.
  • Necesitas revisar registros de correo MX.
  • Estás diagnosticando dominios, hosting o configuración web.

✅ Resultado esperado

Podrás comprobar si un dominio resuelve correctamente, qué servidor DNS respondió y qué direcciones IP o registros devuelve Windows desde la consola.

Errores comunes

nslookup devuelve "Non-existent domain"

El dominio no existe, fue escrito incorrectamente o el servidor DNS no puede resolverlo. Verifica la ortografía antes de continuar.

Request timed out

El servidor DNS no respondió dentro del tiempo esperado. Comprueba tu conexión o prueba con un DNS público como 8.8.8.8 o 1.1.1.1.

Servidor desconocido

Windows no pudo identificar el servidor DNS configurado. Esto puede indicar un problema de configuración de red o un fallo temporal del proveedor de Internet.

Obtengo varias direcciones IP

Es completamente normal. Muchos sitios web utilizan balanceo de carga, CDN o múltiples servidores para mejorar el rendimiento.

Preguntas frecuentes

¿nslookup modifica mi conexión?

No. El comando únicamente consulta servidores DNS y muestra la respuesta recibida. No realiza cambios en la configuración de Windows.

¿Funciona en Windows 11?

Sí. Está disponible tanto en Windows 10 como en Windows 11 y versiones anteriores de Windows.

¿Necesito permisos de administrador?

No. Cualquier usuario puede ejecutar nslookup desde CMD o PowerShell.

¿Cuál es la diferencia entre nslookup y ping?

nslookup consulta el servidor DNS para obtener direcciones IP, mientras que ping comprueba si existe comunicación con una dirección IP o un dominio determinado.
