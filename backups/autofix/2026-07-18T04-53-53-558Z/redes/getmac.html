getmac: descubre la dirección MAC de tu computadora

El comando getmac permite mostrar rápidamente la dirección MAC de todos los adaptadores de red instalados en Windows. Es una herramienta muy utilizada por técnicos, administradores de redes y usuarios que necesitan identificar un dispositivo dentro de una red local.

Aprende a usar getmac

Con este comando podrás identificar la dirección física (MAC Address) de cualquier adaptador de red instalado en Windows.

¿Qué hace el comando getmac?

getmac muestra la dirección MAC (Media Access Control) de los adaptadores de red instalados en Windows. La dirección MAC es un identificador físico único asignado por el fabricante a cada tarjeta de red Ethernet o Wi-Fi.

Este comando resulta especialmente útil para configurar filtros MAC en routers, identificar equipos dentro de una red, realizar inventarios o solucionar problemas de conectividad.

💡 Consejo WinHacks

Muchos routers permiten autorizar o bloquear dispositivos mediante su dirección MAC. Con getmac puedes obtenerla en segundos sin abrir la configuración del sistema.

⚠️

Importante

El comando únicamente consulta la información de los adaptadores de red. No modifica ninguna configuración del sistema ni requiere permisos de administrador.

Paso a paso

1

Abre el Símbolo del sistema

Presiona Win, escribe CMD y abre el Símbolo del sistema. No es necesario ejecutarlo como administrador.

2

Ejecuta el comando getmac

Escribe el siguiente comando y presiona Enter.

getmac
3

Observa la dirección MAC

Windows mostrará la dirección física de cada adaptador de red instalado junto con su estado de funcionamiento.

4

Obtén información más detallada

Puedes mostrar información adicional utilizando el parámetro /v.

getmac /v
5

Muestra el resultado como tabla

Para facilitar la lectura puedes utilizar el formato tabular.

getmac /fo table /v
6

Consulta la configuración completa

Si necesitas más información sobre los adaptadores de red utiliza el siguiente comando.

ipconfig /all

¿Cuándo utilizar getmac?

  • Identificar un equipo dentro de la red.
  • Configurar filtros MAC en el router.
  • Administrar dispositivos Wi-Fi.
  • Realizar inventarios de hardware.

  • Obtener la dirección física de una tarjeta Ethernet o Wi-Fi.
  • Automatizar tareas mediante scripts.

✅ Resultado esperado

Obtendrás la dirección MAC de todos los adaptadores instalados en Windows, junto con información útil para administración, inventario y diagnóstico de redes.

Errores comunes

No aparece ninguna dirección MAC

Comprueba que el adaptador de red esté habilitado. Si el dispositivo está desactivado o no tiene controladores instalados, Windows podría no mostrar información.

Aparecen varias direcciones MAC

Es completamente normal. Windows muestra la dirección MAC de todos los adaptadores instalados, incluyendo Ethernet, Wi-Fi, Bluetooth, adaptadores virtuales y máquinas virtuales.

La dirección MAC cambió

Algunos adaptadores permiten utilizar direcciones MAC aleatorias o personalizadas. También puede cambiar al utilizar determinadas VPN, hipervisores o configuraciones avanzadas del controlador.

El comando no se reconoce

Verifica que escribiste correctamente getmac. El comando está disponible de forma predeterminada en Windows 10, Windows 11 y Windows Server.

Preguntas frecuentes

¿Qué es una dirección MAC?

Es un identificador físico único asignado por el fabricante a cada adaptador de red. Se utiliza para identificar dispositivos dentro de una red local.

¿Es lo mismo que una dirección IP?

No. La dirección MAC identifica el hardware del adaptador de red, mientras que la dirección IP identifica el dispositivo dentro de una red y puede cambiar con el tiempo.

¿Necesito permisos de administrador?

No. getmac puede ejecutarse desde CMD o PowerShell con una cuenta de usuario estándar.

¿Funciona en Windows 11?

Sí. Es compatible con Windows 10, Windows 11 y Windows Server.
