CMD guarda historial de comandos durante la sesión

La consola de Windows puede mostrar comandos usados durante la sesión actual con doskey /history. Es útil para recordar comandos recientes, revisar lo que ejecutaste y trabajar más rápido en CMD.

¿Qué hace doskey /history?

doskey /history muestra los comandos que escribiste en la ventana actual de CMD. No es un historial permanente de todo Windows: normalmente funciona dentro de la sesión abierta.

Esto sirve para recuperar comandos largos, repetir diagnósticos, documentar pasos técnicos o revisar qué ejecutaste sin tener que recordarlo manualmente.

💡 Consejo WinHacks

Si estás haciendo soporte técnico, puedes guardar el historial al final de la sesión para documentar los comandos usados.

🚨

Importante

Este historial puede incluir rutas, nombres de archivos o comandos sensibles. No compartas capturas sin revisar lo que aparece.

Paso a paso

1

Abre CMD

Presiona Win, escribe CMD y abre el Símbolo del sistema.

2

Ejecuta algunos comandos

Escribe algunos comandos de prueba, por ejemplo:

ipconfig
ping google.com
3

Muestra el historial

Ahora escribe:

doskey /history
4

Revisa la lista

CMD mostrará los comandos utilizados durante esa sesión. Puedes copiar alguno y volverlo a ejecutar.

5

Guarda el historial en un archivo

Puedes exportarlo a un archivo de texto con este comando:

doskey /history > historial-cmd.txt

Limitaciones importantes

No es historial permanente

El historial mostrado por doskey /history pertenece normalmente a la ventana actual de CMD. Si cierras CMD, puedes perder ese historial.

No muestra PowerShell

Este comando aplica a CMD. PowerShell maneja su historial de otra forma.

No muestra todo Windows

No registra todo lo que haces en el sistema. Solo muestra comandos escritos en esa sesión de consola.

✅ Resultado esperado

CMD mostrará una lista de comandos ejecutados durante la sesión actual. También podrás guardarla como archivo TXT si necesitas documentarla.

Errores comunes

No aparece nada

Ejecuta primero algunos comandos en esa misma ventana de CMD. Si no hay comandos previos, no habrá historial que mostrar.

Escribí doskey / history y falla

El comando correcto es sin espacio después de la barra: doskey /history.

Cerré CMD y se perdió

Es normal. Si necesitas conservarlo, expórtalo antes con doskey /history > historial-cmd.txt.

Preguntas frecuentes

¿Funciona en Windows 11?

Sí. También funciona en Windows 10.

¿Necesito administrador?

No. Normalmente funciona desde CMD normal.

¿Muestra comandos antiguos?

No necesariamente. Muestra los comandos de la sesión actual de CMD.

¿Sirve para auditoría?

Sirve para revisión básica de la sesión actual, pero no reemplaza herramientas de auditoría profesional.
