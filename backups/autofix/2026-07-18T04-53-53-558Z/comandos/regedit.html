¿Qué es regedit?

regedit es el comando que abre el Editor del Registro de Windows. El Registro funciona como una base de datos donde Windows y muchos programas guardan configuraciones importantes.

Es útil para revisar rutas internas, historial de funciones, claves de programas, configuraciones avanzadas y registros que no aparecen en la interfaz normal.

💡 Consejo WinHacks

Antes de modificar cualquier clave, exporta una copia. Así puedes restaurarla si algo sale mal.

⚠️

No edites claves al azar

El Registro es una zona avanzada. Cambiar o borrar valores incorrectos puede afectar Windows, programas, drivers o el inicio del sistema.

Paso a paso

1

Abre Ejecutar

Presiona Win + R para abrir la ventana Ejecutar.

2

Escribe regedit

Introduce el comando y presiona Enter.

regedit
3

Acepta el aviso de control de cuentas

Windows puede pedir confirmación de administrador. Acepta solo si abriste tú la herramienta.

4

Conoce las claves principales

Verás secciones como HKEY_CURRENT_USER, HKEY_LOCAL_MACHINE y HKEY_CLASSES_ROOT.

5

Exporta una copia antes de modificar

Selecciona una clave, entra en Archivo → Exportar y guarda una copia antes de hacer cambios.

6

Navega con una ruta exacta

Si tienes una ruta específica, pégala en la barra superior del Editor del Registro.

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\RunMRU

Claves importantes del Registro

HKEY_CURRENT_USER

Configuraciones del usuario actual.

HKEY_LOCAL_MACHINE

Configuraciones globales del sistema, hardware, drivers y software instalado.

HKEY_CLASSES_ROOT

Asociaciones de archivos, extensiones y clases del sistema.

HKEY_USERS

Configuraciones de perfiles de usuario cargados en Windows.

¿Cuándo usar regedit?

  • Revisar rutas ocultas de Windows.
  • Consultar claves como RunMRU, USBSTOR o GameBar.
  • Diagnosticar configuraciones avanzadas.
  • Exportar claves antes de cambios técnicos.
  • Seguir guías avanzadas de Windows con rutas exactas.

✅ Resultado esperado

Podrás abrir el Editor del Registro, navegar claves internas y revisar configuraciones avanzadas sin modificar nada por accidente.

Errores comunes

Acceso denegado

Algunas claves requieren permisos elevados o pertenecen al sistema.

No encuentro la ruta

Copia la ruta exacta y pégala en la barra superior del Editor del Registro.

Borré una clave por error

Restaura la copia exportada si la tienes. Si no, usa Restaurar sistema.

Windows se comporta raro después de cambiar algo

Revierte el cambio o usa un punto de restauración del sistema.

Preguntas frecuentes

¿regedit es peligroso?

No por abrirlo. El riesgo aparece al modificar o borrar claves sin saber su función.

¿Funciona en Windows 11?

Sí. También funciona en Windows 10.

¿Necesito administrador?

Para abrirlo normalmente sí puede pedir confirmación UAC. Algunas claves requieren más permisos.

¿Puedo restaurar cambios?

Sí, si exportaste la clave antes o tienes un punto de restauración del sistema.
