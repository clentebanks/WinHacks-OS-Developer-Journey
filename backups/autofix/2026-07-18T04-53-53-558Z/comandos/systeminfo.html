Aprende a usar systeminfo

En este Short descubrirás cómo obtener un reporte completo del sistema utilizando un único comando de Windows.

¿Qué aprenderás?

  • Cómo ejecutar el comando systeminfo.
  • Interpretar la información del sistema.
  • Consultar memoria RAM instalada.
  • Ver versión de Windows y BIOS.
  • Revisar actualizaciones instaladas.
  • Guardar el resultado en un archivo.

¿Qué es systeminfo?

systeminfo es un comando integrado de Windows que recopila información técnica del sistema operativo y del hardware instalado. Es una de las herramientas más utilizadas por administradores de sistemas y técnicos para conocer rápidamente las características de un equipo.

A diferencia de otras utilidades gráficas, este comando puede ejecutarse desde CMD, scripts por lotes o PowerShell, siendo ideal para automatización, inventarios y soporte remoto.

💡 Consejo WinHacks

Si vas a publicar una duda en un foro o pedir ayuda técnica, ejecuta primero systeminfo. El reporte contiene gran parte de la información que normalmente te solicitarán.

⚠️

Importante

El comando únicamente consulta información del sistema. No modifica ninguna configuración ni representa riesgo para Windows.

Paso a paso

1

Abrir el Símbolo del sistema

Presiona Win, escribe CMD y abre el Símbolo del sistema. No es necesario ejecutarlo como administrador.

2

Ejecuta el comando

Escribe el siguiente comando y presiona Enter.

systeminfo
3

Espera el análisis

Windows recopilará información del equipo durante unos segundos.

4

Consulta la información del sistema

Encontrarás datos como versión de Windows, nombre del equipo, procesador, memoria física, BIOS, dominio y arquitectura.

5

Busca las actualizaciones instaladas

Más abajo encontrarás el número de revisiones (Hotfix) instaladas mediante Windows Update.

6

Guardar el informe

Puedes exportar el resultado a un archivo de texto utilizando este comando.

systeminfo > systeminfo.txt

Información que muestra systeminfo

🖥️ Sistema operativo

Nombre del sistema operativo, versión, compilación, arquitectura y fecha de instalación.

💻 Hardware

Procesador, memoria RAM instalada, BIOS, fabricante y modelo del equipo.

🌐 Red

Dominio, nombre del equipo, adaptadores y configuración básica.

🛡️ Actualizaciones

Número de Hotfix instalados mediante Windows Update.

¿Cuándo utilizar systeminfo?

  • Consultar especificaciones del equipo.
  • Ver la versión exacta de Windows.
  • Confirmar memoria RAM instalada.
  • Comprobar la arquitectura del sistema.
  • Revisar información de BIOS.
  • Generar un informe para soporte técnico.
  • Automatizar inventarios mediante scripts.

✅ Resultado esperado

Obtendrás un informe completo del sistema operativo, hardware, memoria, BIOS, actualizaciones y configuración básica del equipo utilizando un solo comando.

Errores comunes

El comando tarda demasiado

Es normal. systeminfo consulta información del sistema, hardware, actualizaciones y configuración antes de mostrar el resultado.

No aparece toda la información

Algunos datos dependen del fabricante del equipo o de la versión de Windows instalada.

No puedo guardar el archivo TXT

Comprueba que tengas permisos de escritura en la carpeta donde estás ejecutando el comando o indica una ruta diferente al exportar.

¿Necesito abrir CMD como administrador?

No. El comando funciona normalmente con una consola estándar.

Preguntas frecuentes

¿systeminfo modifica Windows?

No. Solo consulta información del sistema operativo y del hardware.

¿Funciona en Windows 11?

Sí. También funciona en Windows 10 y en versiones anteriores compatibles.

¿Qué diferencia existe entre systeminfo y msinfo32?

systeminfo muestra un resumen desde CMD, mientras que msinfo32 ofrece una vista gráfica mucho más completa con componentes, recursos y entorno de software.

¿Puedo usar systeminfo en scripts?

Sí. Es uno de los comandos más utilizados por administradores para crear inventarios automáticos y recopilar información de equipos.
