Comandos esenciales

Estos comandos son seguros y útiles para usuarios normales. Úsalos para abrir herramientas integradas de Windows sin perder tiempo buscándolas en menús.

⚙️

services.msc

Abre la consola de servicios de Windows para revisar procesos del sistema.

Leer guía →
🚀

msconfig

Accede a configuración del sistema, inicio selectivo y opciones de arranque.

Leer guía →
🧹

cleanmgr

Abre la limpieza de disco clásica para liberar espacio de forma segura.

Leer guía →
🚨

perfmon /rel

Abre el Historial de Confiabilidad para ver errores ocultos de Windows.

Leer guía →
🛠️

sfc /scannow

Revisa archivos del sistema y ayuda a reparar daños básicos en Windows.

Leer guía →
📊

resmon

Abre el Monitor de Recursos para revisar CPU, memoria, disco y red.

Leer guía →
👁️

eventvwr.msc

Abre el Visor de Eventos para revisar registros y errores avanzados.

Leer guía →
🧬

dxdiag

Revisa información de DirectX, pantalla, sonido y hardware del sistema.

Leer guía →
🖥️

msinfo32

Abre Información del sistema para ver datos completos del equipo.
