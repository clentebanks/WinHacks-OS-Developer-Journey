  • Cuándo usarlo para diagnosticar una PC lenta.

¿Qué hace este comando?

El comando resmon abre el Monitor de Recursos de Windows. Esta herramienta muestra en tiempo real qué procesos están usando CPU, memoria RAM, disco y red.

Es más detallado que el Administrador de tareas porque permite ver actividad por proceso, archivos abiertos, conexiones de red, lectura y escritura de disco.

💡 Consejo WinHacks

Si la PC está lenta, no cierres procesos al azar. Primero revisa qué recurso está saturado: CPU, memoria, disco o red.

⚠️

Antes de continuar

No todos los procesos con nombres raros son peligrosos. Muchos pertenecen a Windows, drivers o programas instalados. Revisa antes de cerrar algo.

Paso a paso

1

Abre Ejecutar

Presiona Win + R para abrir la ventana Ejecutar.

2

Escribe el comando

Escribe este comando y presiona Enter:

resmon
3

Revisa Información general

Windows abrirá el Monitor de Recursos. En la pestaña Información general verás un resumen de CPU, disco, red y memoria.

Captura pendiente: Monitor de Recursos abierto en Información general.

Qué puedes revisar dentro

CPU

Muestra procesos activos, consumo del procesador y servicios asociados.

Memoria

Permite revisar qué programas consumen RAM y cómo se está usando la memoria física.

Disco

Muestra qué procesos están leyendo o escribiendo archivos en el almacenamiento.

Red

Permite ver conexiones activas, procesos con tráfico de red y uso de internet.

Qué debes revisar primero

Si tu PC está lenta, revisa primero qué sección tiene más actividad. Si el disco está al 100%, entra en Disco. Si la memoria está llena, revisa Memoria.

Si un proceso aparece usando muchos recursos todo el tiempo, anota su nombre antes de cerrarlo o investigarlo.

✅ Resultado esperado

Al usar resmon deberías poder identificar qué proceso está consumiendo CPU, memoria, disco o red en tu PC.

Cuándo usar este comando

  • Cuando la PC se pone lenta sin razón.
  • Cuando el disco trabaja al 100%.
  • Cuando la memoria RAM se llena rápido.
  • Cuando el internet se siente lento.
  • Cuando un programa consume demasiado.

Errores comunes

Cerrar procesos sin revisar

Algunos procesos son necesarios para Windows. Antes de cerrar algo, revisa el nombre, la ubicación y el consumo real.

Confundir actividad normal con problema

Windows puede usar disco o red por actualizaciones, antivirus, indexación o sincronización. Revisa si el consumo es constante o temporal.

No combinarlo con otras herramientas

Para un diagnóstico mejor, usa resmon junto con el Administrador de tareas, Historial de Confiabilidad y Visor de Eventos.
