eventvwr.msc: Abre el Visor de Eventos oculto de Windows

Windows registra prácticamente todo lo que sucede en tu computadora. Errores de aplicaciones, apagados inesperados, fallos de drivers, problemas del sistema e incluso advertencias quedan almacenadas dentro del Visor de Eventos. Con eventvwr.msc puedes acceder rápidamente a estos registros para diagnosticar problemas como lo hacen muchos técnicos.

Aprende a usar eventvwr.msc

Este Short muestra cómo abrir el Visor de Eventos de Windows para descubrir errores ocultos que normalmente pasan desapercibidos.

¿Qué aprenderás?

  • Abrir el Visor de Eventos desde Win + R.
  • Encontrar errores críticos del sistema.
  • Interpretar eventos de Windows.
  • Localizar bloqueos de programas.
  • Usar los registros para diagnosticar problemas.

¿Qué es eventvwr.msc?

Es un acceso directo a una de las herramientas administrativas más potentes de Windows. El Visor de Eventos almacena miles de registros generados por el sistema, drivers, aplicaciones y servicios de Windows.

Cuando Windows presenta pantallas azules, errores, programas que se cierran solos o problemas de hardware, normalmente la respuesta queda registrada aquí.

💡 Consejo WinHacks

Si tu PC falla sin explicación aparente, revisa primero el Visor de Eventos antes de reinstalar Windows.

⚠️

Importante

Ver registros no modifica absolutamente nada en Windows. Es una herramienta completamente segura para consultar información del sistema.

Paso a paso

1

Abrir Ejecutar

Presiona Win + R para abrir la ventana Ejecutar de Windows.

2

Escribe eventvwr.msc

Introduce el siguiente comando y presiona Enter.

eventvwr.msc
3

Abre Registros de Windows

En el panel izquierdo despliega Registros de Windows.

4

Selecciona Sistema

Haz clic en Sistema para visualizar errores relacionados con Windows, controladores y servicios.

5

Busca eventos críticos

Revisa especialmente los registros marcados como Crítico, Error o Advertencia.

6

Abre los detalles

Haz doble clic sobre cualquier evento para conocer su descripción, identificador (ID) y origen.

Tipos de eventos más importantes

🔴 Crítico

Fallos graves del sistema, apagados inesperados o errores de hardware.

❌ Error

Problemas de aplicaciones, servicios, controladores o componentes de Windows.

⚠️ Advertencia

Situaciones que podrían convertirse en errores si continúan ocurriendo.

ℹ️ Información

Eventos normales del funcionamiento diario del sistema.

¿Cuándo usar eventvwr.msc?

  • Pantallas azules (BSOD).
  • Reinicios inesperados.
  • Aplicaciones que se cierran solas.
  • Problemas con drivers.
  • Errores después de Windows Update.
  • Diagnóstico de hardware.

✅ Resultado esperado

Podrás consultar el historial completo de eventos de Windows, localizar errores críticos y obtener información para diagnosticar problemas del sistema mucho más rápido.

Errores comunes

No aparecen eventos recientes

Verifica que estés revisando el registro correcto. Los eventos del sistema suelen encontrarse en Registros de Windows → Sistema, mientras que los errores de programas normalmente aparecen en Aplicación.

Hay demasiados registros

Utiliza la opción Filtrar registro actual... para mostrar únicamente eventos Críticos, Errores o Advertencias y facilitar el diagnóstico.

No entiendo el ID del evento

El identificador (Event ID) sirve para buscar información técnica específica. Anota el número y el origen del evento antes de investigarlo.

¿Puedo borrar los registros?

Sí, pero normalmente no es recomendable. Los registros históricos pueden ayudarte a encontrar la causa de problemas futuros.

Preguntas frecuentes

¿eventvwr.msc es seguro?

Sí. Es una herramienta oficial de Microsoft que únicamente muestra registros del sistema. Consultarlos no modifica ninguna configuración.

¿Necesito permisos de administrador?

Para consultar la mayoría de registros no. Algunas acciones administrativas sí pueden requerir privilegios elevados.

¿Funciona en Windows 11?

Sí. El comando eventvwr.msc funciona tanto en Windows 10 como en Windows 11.

¿Puede ayudarme a encontrar la causa de una pantalla azul?

En muchos casos sí. Los eventos críticos y errores del sistema pueden ofrecer pistas sobre fallos de controladores, hardware o apagados inesperados.
