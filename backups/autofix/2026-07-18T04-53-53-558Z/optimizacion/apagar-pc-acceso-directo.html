Mira cómo crear el botón de apagado

En este Short verás cómo crear un acceso directo para apagar tu PC al instante desde el escritorio.

Qué hace este truco de Windows

Este método crea un acceso directo en el escritorio que ejecuta el comando de apagado de Windows. En lugar de abrir el menú Inicio, buscar el botón de energía y confirmar el apagado, puedes hacerlo con un solo clic.

El comando usa la herramienta oficial shutdown de Windows. La opción /s indica que el equipo debe apagarse, y /t 0 indica que el apagado debe comenzar sin espera.

Consejo WinHacks

Después de crear el acceso directo, cambia su icono desde Propiedades. Así evitas confundirlo con otros accesos directos y le das un aspecto más profesional.

Advertencia importante

Este botón apaga el equipo de inmediato. Guarda tus archivos antes de usarlo para evitar perder documentos, proyectos o sesiones abiertas.

Paso a paso para crear el botón de apagado

1

Crea un acceso directo nuevo

Haz clic derecho en el escritorio, entra en Nuevo y selecciona Acceso directo.

2

Escribe el comando de apagado

En el campo de ubicación, copia y pega este comando:

shutdown /s /t 0
3

Asigna un nombre claro

Puedes llamarlo Apagar PC, Apagado rápido o cualquier nombre que te permita reconocerlo fácilmente.

4

Cambia el icono

Haz clic derecho sobre el acceso directo, entra en Propiedades, selecciona Cambiar icono y elige uno relacionado con energía o apagado.

Qué significa el comando

El comando shutdown /s /t 0 usa parámetros simples:

  • shutdown: herramienta oficial para apagar, reiniciar o cerrar sesión.
  • /s: ordena apagar el equipo.
  • /t 0: establece el temporizador en cero segundos.

Resultado esperado

Al hacer doble clic en el acceso directo, Windows iniciará el apagado inmediatamente. Es ideal si quieres reducir clics y tener un botón directo en el escritorio.

Errores comunes

El acceso directo no funciona

Revisa que el comando esté escrito exactamente como shutdown /s /t 0. Si falta un espacio o una barra, puede fallar.

No encuentro un icono adecuado

En la ventana de cambio de icono, Windows puede mostrar iconos del sistema. También puedes buscar iconos dentro de archivos como shell32.dll.

Se apaga demasiado rápido

Eso es normal porque el temporizador está en cero. Si quieres dar unos segundos de margen, cambia el comando a shutdown /s /t 10.

Preguntas frecuentes

¿Este comando es seguro?

Sí. Es un comando oficial de Windows. El riesgo principal es usarlo sin guardar archivos abiertos.

¿Funciona en Windows 10 y Windows 11?

Sí. El comando shutdown funciona en versiones modernas de Windows, incluyendo Windows 10 y Windows 11.

¿Puedo crear otro botón para reiniciar?

Sí. Para reiniciar el equipo puedes usar shutdown /r /t 0, pero úsalo con la misma precaución.
