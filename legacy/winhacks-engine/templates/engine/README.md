# WINHACKS ENGINE v1.0

Este folder contiene los bloques reutilizables oficiales de WinHacks.

No es una carpeta pública para usuarios.

Su función es servir como biblioteca interna para construir páginas, artículos, recursos y módulos sin improvisar HTML.

## Uso

1. Copiar la plantilla necesaria.
2. Reemplazar contenido.
3. Mantener las clases oficiales.
4. No crear nuevos componentes si ya existe uno equivalente.

## Motores incluidos

- Article Engine
- Category Engine
- Resource Engine
- Global Components

## Regla principal

En WinHacks no se diseñan páginas desde cero.

Se ensamblan páginas usando componentes oficiales.
