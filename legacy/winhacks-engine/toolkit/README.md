# WinHacks Toolkit

Punto de entrada unificado para las herramientas técnicas del proyecto.

## Ejecutar todo

```bash
npm run toolkit
```

El comando ejecuta, en este orden:

1. Build Engine
2. RSS Builder
3. Sitemap Builder
4. SEO Auditor
5. Dashboard SEO

El reporte principal queda disponible en `reports/index.html`.

## Comandos individuales

```bash
npm run build
npm run rss
npm run sitemap
npm run seo
npm run seo:raw
npm run seo:dashboard
```
