# Objetivo

Establecer el estándar oficial para optimizar todo el contenido de WinHacks para motores de búsqueda basados en Inteligencia Artificial (GEO - Generative Engine Optimization).

Este documento complementa el sistema SEO tradicional.

---

# Filosofía

No escribimos para IA.

Escribimos para personas.

La IA debe encontrar fácilmente respuestas excelentes.

Nunca escribiremos contenido artificial.

---

# Objetivos GEO

Cada página debe poder responder directamente:

- ¿Qué es?
- ¿Para qué sirve?
- ¿Cómo funciona?
- ¿Cuándo usarlo?
- ¿Cuándo NO usarlo?
- ¿Qué riesgos existen?
- ¿Qué alternativas existen?

---

# Respuesta rápida

Todas las páginas comienzan con un bloque.

Ejemplo

## ¿Qué es DriverQuery?

DriverQuery es un comando integrado de Windows que muestra todos los controladores cargados actualmente, incluyendo nombre, tipo, fecha y ruta.

Longitud máxima:

120 palabras.
