# ¿Qué es WinHacks?

WinHacks es un ecosistema de contenido especializado en Windows cuyo objetivo es convertirse en la biblioteca en español más completa sobre Windows 10 y Windows 11.

No es solamente un canal de YouTube.

No es solamente una página web.

No es solamente una futura Academy.

WinHacks es un sistema donde todos los contenidos trabajan juntos.

```
YouTube Shorts
       │
       ▼
Sitio Web
       │
       ▼
Recursos PDF
       │
       ▼
Newsletter
       │
       ▼
Academy
       │
       ▼
Comunidad
```

Cada contenido debe dirigir al siguiente.

Nunca deben existir contenidos aislados.
