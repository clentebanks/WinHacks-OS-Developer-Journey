Qué hace el comando recent

El comando recent abre una carpeta especial de Windows donde aparecen accesos a archivos utilizados recientemente. Es útil cuando trabajaste en un documento, imagen, PDF o archivo descargado y no recuerdas su ubicación.

Este truco no recupera archivos eliminados. Solo muestra elementos recientes registrados por Windows.

Advertencia de privacidad

Si compartes tu computadora, esta carpeta puede mostrar rastros de archivos usados recientemente. Revisa esta sección si necesitas cuidar tu privacidad.

Paso a paso para ver archivos recientes

1

Abre Ejecutar

Presiona Win + R para abrir la ventana Ejecutar.

2

Escribe el comando

Copia o escribe este comando y presiona Enter:

recent
3

Busca tu archivo

Windows abrirá la carpeta de archivos recientes. Busca el documento, imagen, PDF o archivo que necesitas.

4

Abre su ubicación real

Haz clic derecho sobre el archivo y usa la opción de abrir ubicación si necesitas saber dónde está guardado realmente.

Cuándo sirve este truco

  • Cuando no recuerdas dónde guardaste un archivo.
  • Cuando descargaste algo y no lo encuentras.
  • Cuando cerraste un documento y necesitas abrirlo rápido.
  • Cuando quieres revisar actividad reciente en tu propio equipo.

Resultado esperado

Windows abrirá una lista de archivos recientes. Desde ahí podrás localizar documentos, imágenes, instaladores, PDFs y otros elementos abiertos recientemente en tu PC.

Errores comunes

No aparece el archivo que busco

Puede que el archivo no haya sido registrado como reciente, que esté eliminado o que se haya abierto desde una ubicación no rastreada por Windows.

El comando no abre nada

Revisa que escribiste recent correctamente en la ventana Ejecutar. También puedes intentar reiniciar el Explorador de archivos.

Solo veo accesos directos

Es normal. La carpeta Recent muestra accesos a archivos usados recientemente, no necesariamente los archivos originales.

Preguntas frecuentes

¿Funciona en Windows 11?

Sí. También funciona en Windows 10.

¿Esto recupera archivos borrados?

No. Este método solo muestra archivos recientes. Para recuperar archivos eliminados necesitas revisar la Papelera, copias de seguridad o herramientas de recuperación.

¿Puedo borrar esta lista?

Sí. Puedes eliminar los accesos recientes desde esa carpeta o ajustar la privacidad del Explorador de archivos.
