# Learning Resources

Only verified resources with a clear purpose should be added here. Each resource must include its exact title, publisher, publication year, link, required chapters or timestamps, and the lab where it is applied.

## Selection policy

A resource is accepted when:

- it teaches the version or concepts required by the current module;
- the exact link has been verified;
- it does not duplicate the primary resource without a clear reason;
- it leads directly to a practical deliverable.

The first verified resource set will be added during the Development Environment module.
