Activa el menú contextual clásico en Windows 11

Aprende a recuperar el menú clásico de clic derecho en Windows 11 para ver más opciones directamente, sin tener que presionar Mostrar más opciones cada vez.

¿Qué hace este truco?

Windows 11 cambió el menú de clic derecho por una versión más compacta. Aunque se ve más moderna, muchas opciones quedan ocultas detrás de Mostrar más opciones.

Este truco modifica una clave del Registro del usuario actual para que Windows vuelva a mostrar el menú contextual clásico desde el primer clic derecho.

💡 Consejo WinHacks

Este cambio es ideal si usas herramientas técnicas, compresores, editores, accesos rápidos o funciones que aparecen mejor en el menú clásico.

⚠️

Antes de continuar

Este método usa el Registro de Windows. Copia el comando exactamente como aparece y no modifiques otras claves si no sabes para qué sirven.

Paso a paso

1

Abre Windows Terminal

Haz clic derecho en el botón de Inicio y abre Terminal, PowerShell o Símbolo del sistema.

Captura pendiente: Windows Terminal abierto.
2

Ejecuta el comando para activar el menú clásico

Copia y pega este comando:

reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve

Este comando crea la clave necesaria para restaurar el menú contextual clásico.

Captura pendiente: comando ejecutado correctamente.
3

Reinicia el Explorador de archivos

Para aplicar el cambio sin reiniciar toda la computadora, ejecuta:

taskkill /f /im explorer.exe & start explorer.exe

La barra de tareas puede desaparecer unos segundos y luego volverá a aparecer.

Captura pendiente: Explorador reiniciado.
4

Prueba el clic derecho

Haz clic derecho sobre el escritorio, una carpeta o un archivo. Ahora deberías ver el menú contextual clásico directamente.

Captura pendiente: menú contextual clásico visible.

Comandos completos

Activar menú contextual clásico

Usa este comando para restaurar el menú clásico de clic derecho en Windows 11.

reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve

Reiniciar Explorador de archivos

Usa este comando para aplicar el cambio sin reiniciar Windows completo.

taskkill /f /im explorer.exe & start explorer.exe

Cómo volver al menú moderno

Si quieres regresar al menú contextual moderno de Windows 11, elimina la clave creada con este comando:

reg delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f

Luego reinicia el Explorador de archivos otra vez:

taskkill /f /im explorer.exe & start explorer.exe

✅ Resultado esperado

Al terminar, Windows 11 mostrará el menú contextual clásico directamente al hacer clic derecho sobre archivos, carpetas o el escritorio.

Cuándo usar este truco

  • Cuando quieres ver todas las opciones desde el primer clic.
  • Cuando usas programas que agregan opciones al menú clásico.
  • Cuando trabajas mucho con archivos y carpetas.
  • Cuando prefieres el comportamiento tradicional de Windows 10.
  • Cuando quieres ahorrar clics en tareas repetitivas.

Errores comunes

No reiniciar el Explorador

Si no reinicias el Explorador de archivos, puede parecer que el comando no funcionó. Ejecuta el reinicio o reinicia la computadora.

Copiar mal las llaves del CLSID

La clave contiene símbolos y guiones. Copia el comando completo para evitar errores.

Usar el comando en Windows 10

Este truco está pensado para Windows 11. Windows 10 ya usa el menú clásico de forma predeterminada.

Preguntas frecuentes

¿Este cambio es permanente?

Sí, normalmente se mantiene después de reiniciar. Puedes revertirlo eliminando la clave del Registro con el comando incluido arriba.

¿Necesito permisos de administrador?

Generalmente no, porque el cambio se aplica en HKCU, que corresponde al usuario actual.

¿Esto daña Windows?

No debería dañar Windows si copias el comando correctamente. Solo cambia el comportamiento del menú contextual para el usuario actual.

¿Puedo volver al menú nuevo?

Sí. Usa el comando de reversión y reinicia el Explorador de archivos.

¿Funciona en todas las versiones de Windows 11?

Funciona en muchas instalaciones de Windows 11, aunque Microsoft puede cambiar comportamientos internos en futuras actualizaciones.
