Cambia el icono de tu USB en Windows

Aprende a personalizar una memoria USB en Windows usando un archivo .ico y un archivo autorun.inf para que aparezca con un icono propio en el Explorador de archivos.

¿Qué hace este truco?

Este método permite que una memoria USB aparezca con un icono personalizado en el Explorador de archivos de Windows. Para lograrlo se usa un archivo de icono con extensión .ico y un pequeño archivo llamado autorun.inf.

No cambia el funcionamiento de la USB. Solo modifica la forma en que Windows muestra la unidad cuando está conectada.

💡 Consejo WinHacks

Usa un icono cuadrado en formato .ico para que se vea mejor. Lo ideal es usar tamaños como 256x256 o 128x128.

⚠️

Antes de continuar

No descargues archivos autorun.inf de sitios desconocidos. En este tutorial lo crearás tú mismo, solo con instrucciones visuales para el icono.

Paso a paso

1

Conecta tu memoria USB

Inserta la USB en tu computadora y abre el Explorador de archivos. Entra directamente a la unidad USB.

Captura pendiente: memoria USB abierta en el Explorador.
2

Copia tu icono a la USB

Copia dentro de la USB un archivo de icono con extensión .ico. Para este ejemplo usaremos el nombre:

icono.ico
Captura pendiente: archivo icono.ico dentro de la USB.
3

Crea un archivo de texto

Dentro de la USB, haz clic derecho en un espacio vacío, selecciona Nuevo y luego Documento de texto.

Captura pendiente: crear documento de texto en la USB.
4

Escribe la configuración del icono

Abre el documento de texto y pega este contenido:

[Autorun] Icon=icono.ico

Guarda los cambios antes de cerrar el Bloc de notas.

5

Cambia el nombre del archivo

Renombra el documento de texto y cámbiale el nombre completo a:

autorun.inf

Si Windows pregunta si quieres cambiar la extensión, confirma con Sí.

Captura pendiente: archivo autorun.inf creado.
6

Expulsa y conecta otra vez la USB

Expulsa la memoria USB de forma segura y vuelve a conectarla. Windows debería mostrar el nuevo icono en el Explorador.

Captura pendiente: USB con icono personalizado visible.

Código completo para autorun.inf

Configuración básica

Este es el contenido mínimo para mostrar un icono personalizado en la unidad USB.

[Autorun] Icon=icono.ico

Configuración con nombre de unidad

También puedes agregar una etiqueta para que Windows muestre un nombre personalizado.

[Autorun] Icon=icono.ico Label=WinHacks USB

Opcional: ocultar los archivos

Para que autorun.inf e icono.ico no se vean fácilmente dentro de la USB, puedes ocultarlos desde las propiedades del archivo.

Selecciona ambos archivos, haz clic derecho, entra en Propiedades, marca Oculto y aplica los cambios.

Captura pendiente: opción Oculto en propiedades del archivo.

✅ Resultado esperado

Al terminar, tu memoria USB debería aparecer con un icono personalizado y, si agregaste Label, también con un nombre propio.

Errores comunes

Guardar el archivo como autorun.inf.txt

Este es el error más común. Debe llamarse exactamente autorun.inf, no autorun.inf.txt.

Usar una imagen PNG o JPG

Windows espera un archivo de icono real. Usa formato .ico, no solo una imagen renombrada.

No expulsar la USB

A veces el icono no aparece hasta que expulsas la unidad y la conectas otra vez.

Preguntas frecuentes

¿Esto funciona en Windows 10 y Windows 11?

Sí. Este método puede funcionar en Windows 10 y Windows 11 usando una unidad USB, un archivo .ico y autorun.inf.

¿Es peligroso usar autorun.inf?

El archivo puede ser riesgoso si viene de fuentes desconocidas. En este caso lo estás creando tú mismo solo para indicar el icono de la unidad.

¿Puedo usar cualquier imagen?

No directamente. Lo recomendable es convertir la imagen a formato .ico antes de usarla.

¿Por qué no aparece el icono?

Revisa que el archivo se llame exactamente autorun.inf, que el icono exista dentro de la USB y que el nombre coincida con Icon=icono.ico.

¿Puedo cambiar también el nombre de la USB?

Sí. Puedes usar la línea Label=Nombre de tu USB dentro del archivo autorun.inf.
