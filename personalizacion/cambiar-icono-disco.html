¿Cómo cambiar el ícono del disco C en Windows?

Para cambiar el ícono del disco C en Windows, abre el Registro como administrador y crea la clave DriveIcons dentro de HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer. Luego crea una clave con la letra del disco, por ejemplo C, y dentro una subclave llamada DefaultIcon. En el valor predeterminado coloca la ruta completa de un archivo .ico.

Antes de tocar el Registro

Este tutorial modifica el Registro de Windows. No borres claves que no entiendas. Crea un punto de restauración o exporta la clave antes de hacer cambios. El método es visual, pero una edición incorrecta del Registro puede causar problemas en el sistema.

Clave correcta del Registro

La ruta que se usa para este cambio es la siguiente:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\DriveIcons

💡 Corrección importante

La clave correcta es DriveIcons. No es Drivelcons. La diferencia visual es pequeña, pero si escribes mal el nombre, Windows no aplicará el icono personalizado.

Paso a paso

1

Prepara tu archivo .ico

Primero necesitas un archivo de icono con extensión .ico. Guárdalo en una ruta estable, por ejemplo:

C:\WinHacks\icons\disco.ico

Evita guardarlo en Descargas o en el Escritorio si después vas a moverlo. Si cambias la ruta, Windows dejará de encontrar el icono.

2

Abre el Editor del Registro como administrador

Presiona Win + R, escribe regedit y acepta el aviso de Control de cuentas de usuario.

regedit
3

Ve hasta Explorer

En el Editor del Registro, navega hasta esta ruta:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer

Si la clave DriveIcons no existe, debes crearla manualmente dentro de Explorer.

4

Crea la clave DriveIcons

Haz clic derecho sobre Explorer, selecciona Nuevo > Clave y nómbrala exactamente:

DriveIcons

Respeta la escritura. Windows necesita ese nombre para reconocer la personalización de unidades.

5

Crea la clave de la unidad

Dentro de DriveIcons, crea una nueva clave con la letra del disco que quieres personalizar. Para el disco C, el nombre será:

C

Si quieres cambiar otra unidad, usa su letra real, por ejemplo D, E o F.

6

Crea DefaultIcon

Dentro de la clave C, crea otra clave llamada:

DefaultIcon

La ruta final debe quedar así:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\DriveIcons\C\DefaultIcon
7

Escribe la ruta del icono

Selecciona DefaultIcon, abre el valor (Predeterminado) y escribe la ruta completa del archivo .ico.

C:\WinHacks\icons\disco.ico

Si tu ruta contiene espacios, Windows normalmente la acepta, pero es mejor usar una ruta simple y permanente.

8

Reinicia el Explorador de archivos

Para ver el cambio, reinicia el Explorador desde el Administrador de tareas o reinicia la PC.

✅ Resultado esperado

El disco C aparecerá con el nuevo ícono personalizado en el Explorador de archivos.

Cómo hacerlo rápido con archivo .reg

Si ya sabes lo que haces, puedes crear un archivo .reg con este contenido y ejecutarlo como administrador. Cambia la ruta del icono antes de usarlo.

Windows Registry Editor Version 5.00 [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\DriveIcons\C\DefaultIcon] @="C:\\WinHacks\\icons\\disco.ico"
⚠️

Usa el .reg solo si entiendes el cambio

No ejecutes archivos .reg descargados de sitios desconocidos. Si vas a usar este método, crea tú mismo el archivo y revisa que solo modifique la ruta indicada.

Cómo volver al ícono original

Para revertir el cambio, elimina la clave de la unidad dentro de DriveIcons. Para el disco C, elimina esta clave:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\DriveIcons\C

Después reinicia el Explorador de archivos para que Windows vuelva al ícono predeterminado.

Errores comunes

Escribiste Drivelcons en lugar de DriveIcons

Este es el error más común. La clave correcta usa la palabra Icons, no lcons.

El archivo no es .ico real

Renombrar una imagen .png a .ico no siempre funciona. Usa un archivo .ico válido.

Moviste el icono después de configurarlo

Si Windows no encuentra el archivo en la ruta configurada, el icono puede desaparecer o volver al predeterminado.

No reiniciaste el Explorador

A veces el cambio no aparece hasta reiniciar el Explorador de archivos o reiniciar la computadora.

Preguntas frecuentes

¿Funciona en Windows 11?

Sí. Este método puede funcionar en Windows 11 usando la clave DriveIcons del Registro.

¿Funciona en Windows 10?

Sí. También puede funcionar en Windows 10 con la misma estructura de claves.

¿Necesito permisos de administrador?

Sí. Al modificar HKEY_LOCAL_MACHINE normalmente necesitas permisos de administrador.

¿Esto cambia todos mis archivos?

No. Solo cambia el ícono visual de la unidad en el Explorador de archivos.

¿Puedo cambiar el ícono del disco D o E?

Sí. Crea una clave con la letra correspondiente dentro de DriveIcons.

¿Qué pasa si borro DriveIcons?

Windows dejará de usar los iconos personalizados configurados ahí y volverá a los iconos predeterminados.

¿Es mejor este método o autorun.inf?

Para discos internos, el método del Registro con DriveIcons suele ser más limpio. Para memorias USB, autorun.inf puede ser más práctico, pero debe usarse con cuidado.

¿Puedo usar un icono dentro de un archivo .exe?

Sí, Windows puede usar recursos de icono de algunos archivos ejecutables, pero para evitar errores es más recomendable usar un archivo .ico directo.
