Personalización Windows

Cambiar el icono de una carpeta en Windows

Windows permite personalizar carpetas usando iconos integrados del sistema o archivos .ico. Este ajuste es útil para organizar proyectos, recursos, juegos, herramientas o carpetas importantes con una apariencia más profesional.

Mira cómo cambiar el icono

En este Short puedes mostrar cómo personalizar una carpeta desde Propiedades usando la opción Cambiar icono.

Qué aprenderás

  • Cómo abrir las propiedades de una carpeta.
  • Cómo entrar en la pestaña Personalizar.
  • Cómo usar iconos integrados de Windows.
  • Cómo seleccionar un archivo .ico.
  • Qué hacer si el icono no cambia inmediatamente.

¿Qué necesitas?

Para cambiar el icono de una carpeta puedes usar los iconos incluidos en Windows o un archivo externo con formato .ico. No uses imágenes PNG o JPG directamente, porque Windows normalmente requiere iconos en formato ICO.

💡 Consejo WinHacks

Crea una carpeta llamada Iconos dentro de tus recursos y guarda ahí tus archivos .ico. Así evitas borrar o mover el icono después de aplicarlo.

⚠️

Importante

Si eliminas o mueves el archivo .ico que usaste, Windows puede dejar de mostrar el icono personalizado.

Paso a paso

1

Haz clic derecho en la carpeta

Ubica la carpeta que quieres personalizar, haz clic derecho sobre ella y selecciona Propiedades.

2

Abre la pestaña Personalizar

Dentro de Propiedades, entra en la pestaña Personalizar.

3

Haz clic en Cambiar icono

En la parte inferior, presiona el botón Cambiar icono.

4

Elige un icono integrado

Puedes seleccionar uno de los iconos que Windows muestra por defecto.

5

Usa un icono personalizado

Si tienes un archivo .ico, haz clic en Examinar y selecciona el icono desde tu carpeta de recursos.

6

Aplica los cambios

Haz clic en Aceptar, luego en Aplicar y nuevamente en Aceptar. Si no cambia de inmediato, actualiza el Explorador.

Formatos recomendados

Formato correcto

Usa archivos .ico. Es el formato más compatible para iconos de carpetas en Windows.

Tamaño recomendado

Lo ideal es usar iconos que incluyan tamaños como 16x16, 32x32, 48x48 y 256x256.

No muevas el archivo

Guarda el icono en una ubicación fija. Si lo mueves después, la carpeta puede perder la personalización.

¿Cuándo usar este truco?

  • Organizar carpetas de trabajo.
  • Separar proyectos importantes.
  • Crear carpetas visuales para gaming.
  • Personalizar escritorios tipo hacker.
  • Ordenar recursos de edición, PDFs o scripts.
  • Hacer que Windows se vea más profesional.

✅ Resultado esperado

La carpeta mostrará un icono diferente, ya sea integrado de Windows o personalizado desde un archivo .ico.

Errores comunes

El icono no cambia

Presiona F5 para actualizar el Explorador. También puedes cerrar y volver a abrir la carpeta.

Windows no acepta mi imagen

Verifica que sea un archivo .ico. PNG y JPG no siempre funcionan directamente como iconos de carpeta.

El icono desapareció

Puede ocurrir si moviste o eliminaste el archivo ICO original.

Todas las carpetas se ven igual

Cambia iconos solo en carpetas clave. Si personalizas demasiadas, puede volverse difícil identificar prioridades.

Preguntas frecuentes

¿Funciona en Windows 11?

Sí. También funciona en Windows 10.

¿Puedo usar PNG como icono?

Lo recomendable es convertir el PNG a .ico antes de usarlo.

¿Esto afecta el contenido de la carpeta?

No. Solo cambia la apariencia visual de la carpeta.

¿Puedo volver al icono original?

Sí. Entra nuevamente en Propiedades → Personalizar → Cambiar icono y restaura el icono predeterminado.
